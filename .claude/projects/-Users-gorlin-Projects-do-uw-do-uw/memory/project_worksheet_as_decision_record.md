The worksheet is not just an analysis tool — it IS the decision artifact. The UW's documented decision (posture, rationale, conditions, overrides) lives in the worksheet itself.

**Why:** Multiple audiences consume the worksheet (primary UW, committee, broker, reinsurer). Each needs a different entry point and depth. The worksheet must stand alone as the official record.

**How to apply:**
- Screen HTML and print PDF should have different UX (interactive nav vs. linear reading order)
- Executive summary / decision header must stand alone for secondary readers
- Decision documentation section is a first-class part of the worksheet, not an afterthought
- Reading paths differ by audience: UW gets funnel, secondary readers get executive brief → detail on demand
- Added as WS-05 through WS-08, plan 112-04
