    **Step 3: Create contextual_validator.py** (stages/analyze/contextual_validator.py):

    Module structure (< 250 lines):
    - `RULES_PATH = Path(__file__).parent.parent.parent / "brain" / "config" / "validation_rules.yaml"`
    - `_load_validation_rules() -> list[dict[str, Any]]`: Load from YAML, return empty list if file missing
    - `_signal_matches_pattern(signal_id: str, pattern: str) -> bool`: Split on `|`, use `fnmatch.fnmatch` for each sub-pattern
    - `_resolve_state_path(state: AnalysisState, path: str) -> Any`: Navigate dotted path with getattr chain, auto-unwrap SourcedValue (has `.value` attr and is not str/int/float/bool)
    - `_evaluate_state_check(rule: dict, state: AnalysisState) -> str | None`: Single path/op/value check; ops: `>`, `<`, `>=`, `<=`, `==`, `empty_or_none`. Return formatted annotation or None.
    - `_evaluate_compound(rule: dict, state: AnalysisState) -> str | None`: Evaluate `all` sub-conditions, all must pass. Collect resolved values for annotation template interpolation.
    - `_evaluate_evidence_regex(rule: dict, evidence: str) -> str | None`: Match patterns against evidence text (case-insensitive). Return annotation on first match.
    - `_evaluate_executive_temporal(rule: dict, evidence: str, state: AnalysisState) -> str | None`: Get departures list from `departure_source` path, check if any name appears in evidence (case-insensitive substring match). Return annotation with exec_name and departure_date.
    - `_evaluate_rule(rule: dict, signal_result: dict, state: AnalysisState) -> str | None`: Dispatch by `condition.type` to the appropriate evaluator.
    - `validate_signals(signal_results: dict[str, dict[str, Any]], state: AnalysisState) -> dict[str, int]`: Main entry point. Iterate signal_results, skip non-TRIGGERED. For each TRIGGERED signal, check all rules. Append matching annotations to `result["annotations"]` list. Return summary dict with keys: `rules_evaluated`, `annotations_added`, `signals_checked`. NEVER modify `status` or `threshold_level`.

    All imports: `fnmatch`, `re`, `logging`, `pathlib.Path`, `typing.Any`, `yaml`, `do_uw.models.state.AnalysisState`.
    ZERO signal ID string literals in the Python file.
