       ```jinja2
       {# Facet-driven section rendering -- dispatches to subsection template fragments.
          Used when a facet has subsections defined in its YAML.
          Requires current_facet_id to be set before including this template.
          Phase 56: Financial Health is the first migrated facet. #}

       {% set facet_data = facet_sections[current_facet_id] %}
       {% set facet = facet_data.facet %}

       {# Replicate context variables from the original monolithic template.
          These are required by all subsection fragments. #}
       {% set density = densities.get('financial', {}) %}
       {% set level = density.level|default('CLEAN') if density else 'CLEAN' %}
       {% set fin = financials or {} %}

       <section id="financial-health" class="page-break">
         {{ density_indicator(level) }}
         <h2>Section 3: {{ facet.name }}</h2>

         {# Section narrative (pre-computed) #}
         {% set narr = narratives.financial if narratives else none %}
         {{ section_narrative(narr, ai_generated=true) }}

         {# Dispatch each subsection to its template fragment #}
         {% for sub in facet_data.subsections %}
           {% include sub.template ignore missing %}
         {% endfor %}
       </section>
       ```

       NOTE: The `ignore missing` on the include is a safety net -- if a fragment file is missing, rendering continues without crashing. In practice all 11 fragments must exist.
