**Step 2: Create `tests/test_integration_120.py`**

Write structural verification tests that validate all 3 outputs. The test file should:

1. Find the most recent output for each ticker (`output/{TICKER}/{TICKER}_worksheet.html`)
2. For each ticker, test that:
   - All expected section IDs are present in the HTML (use regex: `<section[^>]*id="([^"]+)"`)
   - v8.0 sections exist: `section-intelligence-dossier`, `section-alternative-data`, `section-forward-looking`, `adversarial-critique`
   - N/A ratio is below 30% (count `<td>` cells containing "N/A" vs total `<td>` cells)
   - Score badge exists (regex: `score-badge|verdict-badge|badge-tier`)
   - SVG chart count is >0 (at least sparklines or charts rendered)
   - File size is >100KB (sanity check — real data produces large output)

3. Per-ticker specific checks:
   - HNGE: scoring section has factor table (F.1 through F.10), litigation section present
   - AAPL: mega-cap, should have extensive financial data tables, peer group
   - ANGI: WALK-tier, should have triggered red flags, litigation content, low score

4. Cross-ticker comparison:
   - `test_angi_differs_from_aapl`: ANGI score should be lower than AAPL score (extract from score-badge text). This validates the "ANGI should look markedly different from AAPL" requirement.

Use `pytest.mark.skipif` to skip individual ticker tests if the output file doesn't exist (graceful degradation if a pipeline run failed).

Pattern from qa_compare.py for HTML parsing:
```python
section_ids = re.findall(r'<section[^>]*id="([^"]+)"', content)
svg_count = len(re.findall(r"<svg", content))
na_cells = len(re.findall(r"<td[^>]*>\s*N/A\s*</td>", content))
total_td = len(re.findall(r"<td", content))
```
