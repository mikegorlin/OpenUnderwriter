**governance_evaluative.py**:
- Board quality metrics: for each metric (independence ratio, avg tenure, meeting attendance, etc.), get the corresponding GOV.* signal do_context
- Governance red flags: for each red flag, get signal do_context
- Pattern: iterate governance signals with `safe_get_signals_by_prefix(signal_results, "GOV.")` and build a dict mapping signal_id -> do_context

**litigation_evaluative.py**:
- SCA cases table: add `do_context` per case from LIT.SCA.* signals
- SEC enforcement: add `do_context` from LIT.SEC.* or REG.SEC.* signals
- If individual case signals don't exist, use the aggregate signal (e.g., LIT.SCA.sca_exposure) for the whole section

**market_evaluative.py**:
- Stock drops: add `do_context` from STOCK.* drop signals
- Insider clusters: add `do_context` from STOCK.INSIDER.* signals
- Executive departures: add `do_context` from GOV.EVENT.* or EXEC.* signals

**scoring_evaluative.py**:
- Scoring factor detail: for each F.1-F.10 factor, collect do_context from contributing signals
- Pattern results: add `do_context` from pattern signals
- Allegation mapping: add `do_context` from allegation signals

- Use the pattern: `sig = safe_get_result(signal_results, SIG_ID); do_ctx = sig.do_context if sig and sig.do_context else ""`
