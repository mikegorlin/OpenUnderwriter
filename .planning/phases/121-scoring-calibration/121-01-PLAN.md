    **Part C: Rewrite apply_crf_ceilings() for weighted compounding**

    Change signature to:
    ```python
    def apply_crf_ceilings(
        composite_score: float,
        red_flag_results: list[RedFlagResult],
        scoring_config: dict[str, Any] | None = None,
        market_cap: float | None = None,
        analysis_results: dict[str, Any] | None = None,
    ) -> tuple[float, str | None]:
    ```

    Keep the old signature working by defaulting new params to None. When scoring_config is None, fall back to the current "lowest ceiling wins" behavior (backward compatible).

    When scoring_config is provided:
    1. Build ceiling_lookup from scoring_config["critical_red_flag_ceilings"]["ceilings"]
    2. For each triggered RedFlagResult, look up its CRF config, call _resolve_crf_ceiling()
    3. Collect all (crf_id, resolved_ceiling, severity_weight) tuples
    4. Sort by ceiling ascending (most severe first)
    5. Primary ceiling = lowest resolved ceiling (from sorted[0])
    6. For each additional CRF beyond the primary, compute: reduction = severity_weight * 0.5 (compounding_factor)
    7. total_reduction = sum of all additional reductions, capped at 0.8 (so primary_ceiling * 0.2 is minimum from compounding)
    8. final_ceiling = max(5, primary_ceiling * (1.0 - total_reduction))
    9. quality_score = min(composite_score, final_ceiling)
    10. Return (quality_score, binding_primary_crf_id)

    Also return ceiling_details as a third element (list of dicts with crf_id, resolved_ceiling, weight, contribution) for logging/debugging. Update the return type annotation to `tuple[float, str | None, list[dict[str, Any]]]`.
