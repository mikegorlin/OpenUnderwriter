    Create `scripts/calibration_baseline.py` that:

    1. **Loads existing state.json files** for each ticker in the calibration set. Look in `output/{TICKER}*/*/state.json` or `output/*{TICKER}*/*/state.json` (glob for the directory pattern). If a ticker has no state.json, print a warning and skip it.

    2. **Computes "before" scores** using the OLD ceiling logic: for each ticker's state, extract composite_score and red_flag_results from state.scoring, then call the old flat-ceiling logic (min of all triggered ceilings) to get the old quality_score and tier.

    3. **Computes "after" scores** using the NEW weighted ceiling logic: call `apply_crf_ceilings()` with `scoring_config`, `market_cap`, and `analysis_results` to get the new quality_score and tier.

    4. **Outputs a comparison table** to stdout:
    ```
    Calibration Baseline Report
    ===========================
    Ticker    | MktCap     | Composite | Before (Score/Tier) | After (Score/Tier) | CRFs | Binding CRF
    ----------|------------|-----------|--------------------|--------------------|------|------------
    AAPL      | $3,600B    | 88.8      | 30.0 / WALK        | 70.0 / WRITE       | 1    | CRF-1
    ANGI      | $0.3B      | 84.5      | 25.0 / WALK        | 12.0 / NO_TOUCH    | 17   | CRF-13
    ...
    ```

    5. **Saves snapshots to JSON**: Write `output/calibration_baseline.json` with structure:
    ```json
    {
      "timestamp": "2026-03-21T...",
      "tickers": {
        "AAPL": {
          "market_cap": 3600000000000,
          "composite": 88.8,
          "before": {"quality_score": 30.0, "tier": "WALK", "binding_crf": "CRF-1"},
          "after": {"quality_score": 70.0, "tier": "WRITE", "binding_crf": "CRF-1", "ceiling_details": [...]},
          "crfs_triggered": 1
        }
      },
      "distribution": {
        "before": {"WIN": 0, "WANT": 0, "WRITE": 0, "WATCH": 0, "WALK": 6, "NO_TOUCH": 0},
        "after": {"WIN": 1, "WANT": 1, "WRITE": 1, "WATCH": 1, "WALK": 1, "NO_TOUCH": 1}
      }
    }
    ```

    6. **Prints distribution summary**:
    ```
    Tier Distribution:
      Before: WALK=6 (100% compressed)
      After:  WIN=1, WANT=1, WRITE=1, WATCH=1, WALK=1, NO_TOUCH=1 (6 distinct tiers)
    ```

    The script should be runnable as: `uv run python scripts/calibration_baseline.py`

    Use the scoring pipeline's own `classify_tier()` function to convert quality scores to tiers, ensuring consistency.

    Important: The script reads existing state.json files -- it does NOT run the full pipeline. This is a scoring-only recalculation using the new ceiling logic.

    For tickers that don't have existing pipeline output (HNGE, EXPO), the script should print a note that they need a pipeline run first and skip them. The baseline can still succeed with 4+ tickers that have output.
