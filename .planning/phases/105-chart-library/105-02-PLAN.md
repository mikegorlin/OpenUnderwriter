The gallery also documents what "great" looks like for each chart type, serving as a visual spec.

    2. Implement pixel comparison utility:
    - Use PIL/Pillow (already in project deps) to load PNG images
    - Compute RMSE between two images: sqrt(mean((pixel_a - pixel_b)^2))
    - Normalize to 0-100 scale (0 = identical, 100 = maximally different)
    - Default threshold: 5% (accounts for anti-aliasing, font rendering differences across platforms)
    - If images differ in size, resize the test image to match golden reference dimensions

    ```python
    def compute_image_rmse(img_a_path: Path, img_b_path: Path) -> float:
        """Compute normalized RMSE between two PNG images."""
        from PIL import Image
        import numpy as np

        a = np.array(Image.open(img_a_path).convert("RGB"), dtype=np.float32)
        b = np.array(Image.open(img_b_path).convert("RGB"), dtype=np.float32)

        if a.shape != b.shape:
            # Resize b to match a
            b_img = Image.open(img_b_path).convert("RGB").resize((a.shape[1], a.shape[0]))
            b = np.array(b_img, dtype=np.float32)

        rmse = np.sqrt(np.mean((a - b) ** 2))
        return float(rmse / 255.0 * 100)  # Normalize to 0-100
    ```
