**Create new file `brain_schema.py` with 3 schema classes:**

5. `PatternDefinition` -- for future Phase 109 pattern YAML files. Based on roadmap requirements (PAT-06):
   ```python
   class PatternDefinition(BaseModel):
       model_config = ConfigDict(extra="forbid")
       id: str  # e.g. "desperate_growth_trap"
       name: str  # Human-readable
       description: str
       required_signals: list[str]  # Signal IDs that must fire
       minimum_matches: int = Field(default=3, ge=1)
       recommendation_floor: str | None = None  # Minimum decision tier when pattern fires
       rap_dimensions: list[Literal["host", "agent", "environment"]]  # Which H/A/E dimensions involved
       historical_cases: list[str] = Field(default_factory=list)  # Case references
       epistemology: Epistemology | None = None
   ```

6. `ChartTemplate` -- matches existing chart_registry.yaml structure but now enforced:
   ```python
   class ChartTemplate(BaseModel):
       model_config = ConfigDict(extra="forbid")
       id: str
       name: str
       module: str  # Python module path
       function: str  # Function name
       params: dict[str, Any] = Field(default_factory=dict)
       signals: list[str] = Field(default_factory=list)  # Signal IDs this chart visualizes
       formats: list[Literal["html", "pdf"]]
       data_requires: list[str] = Field(default_factory=list)
       section: str
       position: int
       call_style: Literal["standard", "minimal", "radar", "ownership", "timeline"]
       overlays: list[str] = Field(default_factory=list)
       # v7.0 additions
       style_overrides: dict[str, Any] = Field(default_factory=dict)
       golden_reference: str | None = Field(default=None, description="Path to golden reference image")
   ```

7. `SeverityAmplifier` -- for Phase 108 severity model:
   ```python
   class SeverityAmplifier(BaseModel):
       model_config = ConfigDict(extra="forbid")
       id: str  # e.g. "media_notoriety"
       name: str
       description: str
       multiplier: float = Field(ge=1.0, le=5.0)  # Severity multiplier
       trigger_condition: str  # When this amplifier activates
       signal_ids: list[str] = Field(default_factory=list)  # Signals that can trigger
       rap_class: Literal["host", "agent", "environment"]
       epistemology: Epistemology
   ```
