```python
def build_template_context(
    state: AnalysisState,
    chart_dir: Path | None = None,
) -> dict[str, Any]:
    """Build the template context dictionary from AnalysisState.
    Returns dict with keys: ticker, gen_date, chart_dir, company_name,
    company, executive_summary, financials, market, governance, litigation,
    scoring, ai_risk, meeting_questions, classification, hazard_profile,
    risk_factors, forensic_composites, executive_risk, temporal_signals,
    nlp_signals, peril_map, narratives, densities, financial_narrative,
    market_narrative, insider_narrative, governance_narrative,
    litigation_narrative, company_narrative, scoring_narrative.
    """
```

Current section signatures (all follow this pattern):
```python

```

New section signatures after migration:
```python
def render_section_N(doc: Any, context: dict[str, Any], ds: DesignSystem, **kwargs) -> None:
```

Context dict key mapping (state -> context):
- state.company -> context["company"]
- state.company.identity -> context["company"] (nested fields)
- state.executive_summary -> context["executive_summary"]
- state.extracted.financials -> context["financials"]
- state.extracted.market -> context["market"]
- state.extracted.governance -> context["governance"]
- state.extracted.litigation -> context["litigation"]
- state.scoring -> context["scoring"]
- state.analysis -> context (classification, hazard_profile, risk_factors, etc.)
- state.benchmark -> accessed via context where applicable
- state.acquired_data -> may need to be ADDED to context if sections need it
