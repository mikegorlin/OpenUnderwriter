    2. Create src/do_uw/stages/extract/stock_drop_decay.py (~60 lines):
       - HALF_LIFE_DAYS = 180
       - DECAY_LAMBDA = math.log(2) / HALF_LIFE_DAYS  (yields ~0.003851)
       - compute_decay_weight(drop_date_str: str, reference_date: date | None = None) -> float
         - reference_date defaults to date.today() for production, injectable for tests
         - Formula: exp(-DECAY_LAMBDA * max(days_ago, 0))
         - Returns 0.0 on invalid/None input
       - compute_decay_weighted_severity(drop_pct: float, decay_weight: float) -> float
         - Returns abs(drop_pct) * decay_weight
       - apply_decay_weights(drops: list[StockDropEvent], reference_date: date | None = None) -> list[StockDropEvent]
         - Sets decay_weight and decay_weighted_severity on each drop
         - Returns drops sorted by decay_weighted_severity descending

    3. Create src/do_uw/stages/extract/stock_drop_decomposition.py (~80 lines):
       - decompose_drop(drop, company_prices, spy_prices, sector_prices, dates) -> StockDropEvent
         - Find drop date index in dates list
         - For multi-day drops: extract prices at start_idx and start_idx + period_days (2-point window)
         - For single-day: extract prices at idx-1 and idx (2-point window)
         - Call compute_return_decomposition with the price windows
         - Set market_pct, sector_pct, company_pct from result
         - Set is_market_driven = True if market contribution accounts for >50% of the total absolute drop
         - When sector data missing or decomposition returns None, leave fields as None
       - decompose_drops(drops, company_prices, spy_prices, sector_prices, dates) -> list[StockDropEvent]
         - Iterates drops, calls decompose_drop on each, skips drops without date
