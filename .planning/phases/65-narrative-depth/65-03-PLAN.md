# Plan 65-03: SCR Framework + D&O Implications Callout Boxes

## Objective
Add SCR (Situation-Complication-Resolution) narrative framework and D&O-specific implications callout boxes to every section of the HTML worksheet.

## Context
- @REQUIREMENTS.md NARR-04: SCR framework per section
- @REQUIREMENTS.md NARR-06: D&O-specific implications callout box in every section

## Tasks

### Task 1: Create SCR + D&O implications Jinja2 macros
type="auto"

Add `scr_narrative` and `do_implications` macros to the existing `components/callouts.html.j2` template. The SCR macro renders a structured Situation-Complication-Resolution block. The D&O implications macro renders a navy-bordered callout box with D&O-specific risk context.

**Done criteria:**
- `scr_narrative(situation, complication, resolution)` macro in callouts.html.j2
- `do_implications(items, section_name)` macro in callouts.html.j2
- CSS classes for `.scr-block` and `.do-implications` in components.css
- Both macros imported in base.html.j2

### Task 2: Create SCR + D&O implications context builder
type="auto"

Create a `narrative.py` context builder module that generates per-section SCR narratives and D&O implications from the existing analysis state. Uses brain section metadata + analysis data to derive situation/complication/resolution for each section, and maps section findings to D&O coverage implications.

**Done criteria:**
- `context_builders/narrative.py` with `extract_scr_narratives()` and `extract_do_implications()` functions
- Functions derive SCR from density levels, check results, and narrative text
- D&O implications map section signals to coverage-relevant observations
- Module under 500 lines
- Integrated into `build_html_context()` in html_renderer.py

### Task 3: Add SCR + D&O implications to section templates
type="auto"

Update the 8 main section templates (company, financial, governance, litigation, market, scoring, executive, ai_risk) to include SCR narrative blocks and D&O implications callout boxes. Uses the context data produced by Task 2.

**Done criteria:**
- All 8 section templates include `{% if scr_narratives.get('section_id') %}` SCR block
- All 8 section templates include `{% if do_implications.get('section_id') %}` callout box
- SCR appears after section heading, before narrative
- D&O implications appears after narrative, before facets
- Graceful degradation when data is absent

### Task 4: Add tests for SCR and D&O implications
type="auto"

Write tests for the narrative context builder functions and verify template rendering with SCR/implications data.

**Done criteria:**
- Tests for `extract_scr_narratives()` with various density levels
- Tests for `extract_do_implications()` with signal mappings
- Tests verify graceful handling of empty/missing data
- All tests pass with `uv run pytest`

## Verification
- `uv run pytest` passes
- No source file over 500 lines
- SCR blocks render when section has density data
- D&O implications render when section has relevant signals

## Success Criteria
- NARR-04: SCR framework applied per section
- NARR-06: D&O implications callout in every section
