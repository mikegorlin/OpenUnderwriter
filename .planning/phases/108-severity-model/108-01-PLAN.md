Implement the full severity computation engine: damages estimation, settlement regression, severity amplifiers, layer erosion, defense costs, and dual ABC/Side A models. Wrap legacy severity as comparison lens.

Purpose: This is the S in P x S. The scoring model (Phase 107) provides P (probability). This plan computes S (severity) -- how bad a claim would be if it happens. Together they produce expected loss, the core risk metric for pricing.
