**1. Create `src/do_uw/stages/render/context_builders/forward_risk_map.py`:**

Function: `extract_forward_risk_map(state: AnalysisState, signal_results: dict[str, Any]) -> dict[str, Any]`

Returns dict with keys:
- `"forward_statements"`: list of dicts, each with: `metric_name`, `current_value`, `guidance_claim`, `guidance_type`, `miss_risk`, `miss_risk_rationale`, `sca_relevance`, `source`, `confidence`, `row_class` (CSS class: "risk-high" / "risk-medium" / "risk-low" based on miss_risk)
- `"forward_statement_count"`: int
- `"has_forward_statements"`: bool
- `"has_quantitative_guidance"`: bool (True if any statement is QUANTITATIVE)
- `"catalysts"`: list of dicts, each with: `event`, `timing`, `impact_if_negative`, `litigation_risk`, `row_class`
- `"catalyst_count"`: int
- `"has_catalysts"`: bool
- `"growth_estimates"`: list of dicts, each with: `period`, `metric`, `estimate`, `trend`, `trend_icon` (arrow emoji: up/down/flat), `source`
- `"has_growth_estimates"`: bool
- `"alt_signals"`: dict with keys for short_interest, analyst_sentiment, buyback data from `state.extracted.market`:
  - `"short_interest"`: dict with `shares_short`, `short_ratio`, `trend` (from ShortInterestProfile if available)
  - `"analyst_sentiment"`: dict with `consensus`, `target_mean`, `coverage_count` (from AnalystSentimentProfile)
  - `"buyback_support"`: dict with `has_buyback`, `amount` (from capital_markets extraction)
  - `"has_alt_signals"`: bool
- `"forward_available"`: bool (True if any forward data exists)

Read from `state.forward_looking.forward_statements`, `.catalysts`, `.growth_estimates`. Read alt signals from `state.extracted.market`.
