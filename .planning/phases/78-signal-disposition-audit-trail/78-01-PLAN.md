3. **SignalDisposition** (Pydantic BaseModel):
   - signal_id: str
   - signal_name: str
   - disposition: DispositionTag
   - skip_reason: SkipReason | None = None (only populated when disposition is SKIPPED)
   - skip_detail: str = "" (human-readable explanation)
   - section_prefix: str = "" (first segment of signal_id, e.g. "FIN" from "FIN.LIQ.position")
   - evidence: str = "" (from SignalResult.evidence when TRIGGERED)

4. **DispositionSummary** (Pydantic BaseModel):
   - total: int
   - triggered_count: int
   - clean_count: int
   - skipped_count: int
   - inactive_count: int
   - dispositions: list[SignalDisposition]
   - by_section: dict[str, dict[str, int]] (section_prefix -> {triggered, clean, skipped, inactive})

5. **build_dispositions(all_signals, signal_results) -> DispositionSummary**:
   - `all_signals`: list[dict] from brain loader (every signal in the brain)
   - `signal_results`: dict[str, Any] from state.analysis.signal_results (evaluated results)
   - For each signal in all_signals:
     a. If lifecycle_state == "inactive" or "deprecated" -> INACTIVE
     b. If signal_id in signal_results -> derive from result status:
        - TRIGGERED -> TRIGGERED
        - CLEAR or INFO -> CLEAN
        - SKIPPED -> SKIPPED with reason derived from data_status/data_status_reason
     c. If signal_id NOT in signal_results:
        - If type == "foundational" -> SKIPPED/FOUNDATIONAL
        - If execution_mode != "AUTO" -> SKIPPED/NOT_AUTO_EVALUATED
        - Otherwise -> SKIPPED/EXTRACTION_GAP (signal exists but wasn't evaluated)
   - Build per-section counts from section_prefix
   - Return DispositionSummary
