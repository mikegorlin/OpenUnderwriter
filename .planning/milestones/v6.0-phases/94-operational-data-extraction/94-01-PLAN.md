```python
def extract_subsidiary_structure(
    state: AnalysisState,
) -> tuple[SourcedValue[dict[str, Any]] | None, ExtractionReport]:
```
- Reuse existing Exhibit 21 parsing from `profile_helpers.py` (call `extract_geographic_footprint` or read from `state.company.geographic_footprint` if already populated)
- Build jurisdiction-level summary: `{"total_subsidiaries": int, "jurisdiction_count": int, "jurisdictions": [{"name": str, "count": int, "regulatory_regime": "HIGH_REG"|"MEDIUM_REG"|"LOW_REG"}], "high_reg_count": int, "low_reg_count": int, "tax_haven_count": int}`
- Classify regulatory regimes using a dict mapping: HIGH_REG jurisdictions = {"united kingdom", "singapore", "hong kong", "china", "india", "germany", "france", "japan", "switzerland", "australia", "south korea", "brazil"} (financial/data regulation heavy), LOW_REG = tax havens from existing `_load_tax_havens()` in profile_helpers.py, everything else = MEDIUM_REG
- Source = "Exhibit 21", confidence = HIGH

```python
def extract_workforce_distribution(
    state: AnalysisState,
) -> tuple[SourcedValue[dict[str, Any]] | None, ExtractionReport]:
```
- First try LLM extraction: call `convert_workforce_distribution()` from converters
- Fallback: regex on 10-K Item 1 text for patterns like "approximately X,XXX employees in the United States" or "XX% of our employees are covered by collective bargaining"
- If only total employee count is available (from `state.company.employee_count`), still produce a result with just total + Nones for breakdown
- Source = "10-K Item 1 (LLM)" or "10-K text", confidence = MEDIUM

```python
def extract_operational_resilience(
    state: AnalysisState,
) -> tuple[SourcedValue[dict[str, Any]] | None, ExtractionReport]:
```
- First try LLM extraction: call `convert_operational_resilience()` from converters
- Derive geographic concentration score (0-100) from geographic_footprint data: if top 1 jurisdiction has >60% of subsidiaries, score = 80+; if top 2 have >80%, score = 60-80; else score proportional to HHI
- Combine: `{"geographic_concentration_score": int, "primary_geography": str, "single_facility_risk": bool, "supply_chain_depth": str, "supply_chain_description": str, "overall_assessment": "STRONG"|"ADEQUATE"|"WEAK"}`
- Overall assessment: WEAK if geo score > 80 or single_facility, STRONG if geo < 40 and supply deep, else ADEQUATE
- Source mixed, confidence = MEDIUM
