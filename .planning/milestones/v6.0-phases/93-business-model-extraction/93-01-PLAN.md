**Part B: Add converter functions in ten_k_converters.py**

Add these converter functions (following the existing pattern of `convert_business_description`, `convert_employee_count`, etc.):

1. `convert_revenue_model_type(ten_k: TenKExtraction) -> SourcedValue[str] | None`: Returns SourcedValue wrapping `ten_k.revenue_model_type` if present, source="10-K Item 1 (LLM)", confidence=MEDIUM.

2. `convert_key_person_risk(ten_k: TenKExtraction) -> SourcedValue[dict[str, Any]] | None`: Combines `is_founder_led`, `ceo_tenure_years`, `has_succession_plan` into a dict. Computes `risk_score` (0-3): +1 if founder-led, +1 if tenure > 10yr, +1 if no succession plan. Returns SourcedValue with the dict. Source="10-K + Proxy (LLM)", confidence=MEDIUM.

3. `convert_segment_lifecycle(ten_k: TenKExtraction) -> list[SourcedValue[dict[str, str | float]]]`: Parses `segment_lifecycle_stages` strings (format "Segment Name: STAGE (X% YoY)") into dicts with keys `name`, `stage`, `growth_rate`. Returns list of SourcedValue.

4. `convert_disruption_risk(ten_k: TenKExtraction) -> SourcedValue[dict[str, Any]] | None`: Takes `disruption_threats` list. Level = HIGH if >= 3 threats, MODERATE if >= 1, LOW if 0. Returns dict with `level`, `threats`, `threat_count`.

5. `convert_segment_margins(ten_k: TenKExtraction) -> list[SourcedValue[dict[str, str | float]]]`: Parses `segment_margins` strings (format "Name: X% (prior year: Y%)") into dicts with `name`, `margin_pct`, `prior_margin_pct`, `change_bps` (computed as (current - prior) * 100).
