   - Check if any offering in `capital_markets.offerings` has `is_window_active(offering.section_11_window_end)` = True
   - Compute `event_ipo_exposure_score` as the count of active windows (0 = clear)

3. **BIZ.EVENT.restatements**: Read from `extracted.financials.audit` if available:
   - `restatements` list/count
   - Also check `analysis.xbrl_forensics` for material weakness flags
   - Compute `event_restatement_severity` as integer: 2 for restatement, 1 for material weakness only, 0 for clean

4. **BIZ.EVENT.capital_changes**: Read from `extracted.market.capital_markets`:
   - Count offerings (debt/equity events)
   - Also check for 8-K Item 2.03 (creation of direct financial obligation) from filing metadata
   - Compute `event_capital_change_count` as integer count of material capital events

5. **BIZ.EVENT.business_changes**: Read from `company.business_changes`:
   - Count entries in `company.business_changes` list

   - Compute `event_business_change_count` as integer
