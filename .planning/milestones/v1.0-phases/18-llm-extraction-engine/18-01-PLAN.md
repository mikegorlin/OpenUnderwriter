       - Import guarding pattern (from pricing_ingestion.py):

         ```python
         try:
             import anthropic
             import instructor as instructor
         except ImportError:
             anthropic = None  # type: ignore[assignment]
             instructor = None  # type: ignore[assignment]
         ```
       - Constructor: `model: str = "anthropic/claude-haiku-4-5"`, `max_retries: int = 2`, `cache: ExtractionCache | None = None`, `budget_usd: float = 1.0`.
       - Creates internal `CostTracker(budget_usd=budget_usd)`.
       - `schema_hash(model: type[BaseModel]) -> str` module-level function: Takes `model.model_json_schema()`, serializes with `json.dumps(sort_keys=True, separators=(",", ":"))`, returns `hashlib.sha256(...).hexdigest()[:12]`.

       - Main method: `extract(filing_text: str, schema: type[T], accession: str, form_type: str, system_prompt: str, max_tokens: int = 8192) -> T | None`:
         1. If `instructor` is None or `anthropic` is None, log warning and return None.
         2. If `ANTHROPIC_API_KEY` not in env, log warning and return None.
         3. If `self._cost_tracker.is_over_budget()`, log warning and return None.
         4. Compute `schema_version = schema_hash(schema)`.
         5. Check cache: if hit, return `schema.model_validate_json(cached_json)`.
         6. Call `strip_boilerplate(filing_text)`.
         7. Quick token estimate with `estimate_tokens()`. If > 190_000, log warning and return None (filing too large even after stripping).
         8. Create instructor client: `instructor.from_provider(self._model)`.
         9. Call `client.chat.completions.create(messages=[system, user], response_model=schema, max_tokens=max_tokens, max_retries=self._max_retries)`.
         10. Cast result to `T`.
         11. Record cost: extract `usage` from the API response. NOTE: instructor returns the Pydantic model directly, not a raw response with usage. Token counts may need to be estimated from the text length and result size. Use `estimate_tokens(filing_text)` for input and `estimate_tokens(result.model_dump_json())` for output as approximations.
         12. Cache result: `self._cache.set(accession, form_type, schema_version, result.model_dump_json(), input_tokens, output_tokens, cost, self._model)`.
         13. Return result.

       - Wrap the API call in try/except. On any exception, log warning and return None (fallback to regex will happen at caller level).

       - Property: `cost_summary -> dict[str, Any]` delegates to `self._cost_tracker.summary()`.
