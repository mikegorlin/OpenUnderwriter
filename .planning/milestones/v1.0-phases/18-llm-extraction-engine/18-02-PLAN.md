<objective>
Create comprehensive Pydantic extraction schemas for every SEC filing type the system acquires, plus system prompts that instruct the LLM on D&O-relevant data extraction. Each schema defines the complete structured output for a single API call per filing document.

Purpose: These schemas are the "what to extract" specification. The LLMExtractor (Plan 01) is the "how to extract" engine. Together they enable complete filing extraction.
Output: Schema files for 10-K, DEF 14A, 10-Q, 8-K, SC 13D/13G, S-3/S-1/424B, plus 20-F/6-K (FPI equivalents), with a registry mapping form types to schemas.
</objective>

    1. **schemas/common.py** (~150 lines): Shared sub-models used across filing schemas. Keep flat (no nesting beyond 2 levels). All fields optional with defaults (LLM may not find everything).
       - `class ExtractedPerson(BaseModel)`: name (str), title (str | None), age (int | None), tenure_years (float | None). Used for directors, officers.
       - `class MoneyAmount(BaseModel)`: amount (float | None), currency (str = "USD"), period (str | None, e.g. "FY2025").
       - `class ExtractedLegalProceeding(BaseModel)`: case_name (str), court (str | None), filing_date (str | None), allegations (str | None), status (str | None), settlement_amount (float | None), class_period_start (str | None), class_period_end (str | None), source_passage (str = Field(description="Exact quote, max 200 chars")).
       - `class ExtractedRiskFactor(BaseModel)`: title (str), category (str = Field(description="LITIGATION, REGULATORY, FINANCIAL, OPERATIONAL, CYBER, ESG, AI, OTHER")), severity (str = Field(description="HIGH, MEDIUM, LOW")), is_new_this_year (bool = False), source_passage (str).
       - `class ExtractedDirector(BaseModel)`: name (str), age (int | None), independent (bool | None), tenure_years (float | None), committees (list[str] = Field(default_factory=list)), other_boards (list[str] = Field(default_factory=list)), source_passage (str = "").
       - `class ExtractedCompensation(BaseModel)`: name (str), title (str | None), salary (float | None), bonus (float | None), stock_awards (float | None), option_awards (float | None), non_equity_incentive (float | None), other_comp (float | None), total_comp (float | None), source_passage (str = "").

       - **Item 1 (Business)**: business_description (str | None), revenue_segments (list[str]), geographic_regions (list[str]), employee_count (int | None), customer_concentration (list[str] -- "Top customer is X% of revenue"), supplier_concentration (list[str]), competitive_position (str | None), regulatory_environment (str | None), is_dual_class (bool | None), has_vie (bool | None).
       - **Item 1A (Risk Factors)**: risk_factors (list[ExtractedRiskFactor]) -- max 25 most significant. D&O-relevant risk factors are the priority.
       - **Item 3 (Legal Proceedings)**: legal_proceedings (list[ExtractedLegalProceeding]).
       - **Item 5 (Market Info)**: stock_exchange (str | None), share_repurchase_amount (float | None).
       - **Item 7 (MD&A)**: revenue_trend (str | None -- "growing/declining/stable with percentage"), margin_trend (str | None), key_financial_concerns (list[str]), critical_accounting_estimates (list[str]), guidance_language (str | None -- any forward-looking statements), non_gaap_measures (list[str]).
       - **Item 7A (Market Risk)**: interest_rate_risk (str | None), currency_risk (str | None), commodity_risk (str | None).
       - **Item 8 (Financial Statements -- footnote highlights only, not the numbers)**: going_concern (bool = False), going_concern_detail (str | None), material_weaknesses (list[str]), debt_instruments (list[str] -- key debt with terms), credit_facility_detail (str | None), covenant_status (str | None), contingent_liabilities (list[str]), tax_rate_notes (str | None), stock_comp_detail (str | None).
       - **Item 9A (Controls)**: has_material_weakness (bool = False), material_weakness_detail (list[str]), significant_deficiencies (list[str]), remediation_status (str | None), auditor_attestation (str | None).
       - **Items 10-14 (Governance)**: auditor_name (str | None), auditor_tenure_years (int | None), auditor_opinion_type (str | None -- "unqualified/qualified/adverse/disclaimer"), related_party_transactions (list[str]).
       - **Filing metadata (extracted from document header)**: fiscal_year_end (str | None), period_of_report (str | None).

       Use `lambda: []` for list default_factory (pyright strict compliance per MEMORY.md).

    3. **schemas/def14a.py** (~350 lines): `DEF14AExtraction` -- comprehensive schema for DEF 14A proxy statements.
