## Decisions Made
- Tailwind v4 uses `@source` directive in input.css for content scanning (v3's `--content` CLI flag is removed in v4)
- Fonts are base64-embedded in compiled CSS (not relative URLs) because Playwright renders from `file://` temp files where relative font paths would not resolve
- compiled.css is committed to the repo as a deterministic build artifact -- it only changes when templates or input.css change
- The Tailwind supplement utility classes in styles.css (bg-bg-alt, bg-gold-light, text-navy, etc.) are now generated by Tailwind's @theme block; the old manual overrides are replaced with a comment noting they're in compiled.css
