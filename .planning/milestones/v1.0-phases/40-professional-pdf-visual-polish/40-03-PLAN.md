  <action>
  1. Add new formatting functions to `formatters.py`:

     `format_currency_accounting(value, compact=False)` -> str:
     - None -> gray italic N/A: `'<span class="text-gray-400 italic">N/A</span>'`
     - Negative values in red parentheses: `'<span class="text-risk-red">(${abs:,.0f})</span>'`
     - For compact=True, use adaptive precision: $(1.2B), $(345M)
     - Positive values: normal `$1,234,567` or `$394.3B` (compact)
     - This replaces format_currency for HTML rendering (keep format_currency for Word/plain text)

     `format_adaptive(value, unit=None)` -> str:
     - Auto-detect and format: ratios to 2 decimals (1.07x), percentages to 1 decimal (23.4%), large numbers in B/M/K notation
     - If unit is 'currency': use format_currency_accounting(compact=True)
     - If unit is 'pct': format_percentage(value, 1)
     - If unit is 'ratio': f"{value:.2f}x"
     - If no unit: auto-detect from magnitude

     `format_yoy_html(change_pct)` -> str:
     - Returns HTML with colored arrow + change:
     - Positive: `'<span class="text-risk-green">&#9650; +12.3%</span>'` (green up-triangle, per LOCKED decision: "triangle-up +12.3% in green")
     - Negative: `'<span class="text-risk-red">&#9660; -5.1%</span>'` (red down-triangle)
     - Zero/None: `'<span class="text-gray-400">--</span>'`
     - The triangle characters: &#9650; (up) &#9660; (down)
     - IMPORTANT: text-risk-green must be defined in input.css @theme (--color-risk-green: #16A34A) and in styles.css as a fallback class

     `format_na(value)` -> str:
     - None -> `'<span class="text-gray-400 italic">N/A</span>'`
     - Otherwise -> str(value)
     - For consistent gray italic N/A across all templates

  2. Add tests in `tests/stages/render/test_formatters.py`:
     - test_format_currency_accounting_negative: -1234 -> `<span class="text-risk-red">($1,234)</span>`
     - test_format_currency_accounting_positive: 1234 -> `$1,234`
     - test_format_currency_accounting_compact: 394_328_000_000 -> `$394.3B`
     - test_format_currency_accounting_none: None -> contains `N/A`
     - test_format_yoy_html_positive: 12.3 -> contains `&#9650;` and `+12.3%` and `text-risk-green`
     - test_format_yoy_html_negative: -5.1 -> contains `&#9660;` and `-5.1%`
     - test_format_adaptive_ratio: (1.07, 'ratio') -> `1.07x`

  3. Register all new filters in `html_renderer.py` `_render_html_template()`:
     ```python

     env.filters["format_adaptive"] = format_adaptive
     env.filters["yoy_arrow"] = format_yoy_html
     env.filters["format_na"] = format_na
     ```
     Import the new functions at the top of html_renderer.py.
