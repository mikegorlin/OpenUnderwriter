  2. Add humanize_enum and strip_markdown as Jinja2 filters in `html_renderer.py`:

     Import `humanize_enum` from formatters.py (already exists).

     Create a `strip_markdown(text)` function:
     ```python
     import re
     def strip_markdown(text: str) -> str:
         """Strip markdown formatting from text for clean HTML display."""
         if not text:
             return ""
         # Remove bold markers: **text** -> text
         text = re.sub(r'\*\*(.+?)\*\*', r'\1', text)
         # Remove italic markers: *text* -> text
         text = re.sub(r'\*(.+?)\*', r'\1', text)
         # Remove markdown headers: ## text -> text
         text = re.sub(r'^#{1,6}\s+', '', text, flags=re.MULTILINE)
         return text.strip()
     ```

     Register both filters:
     ```python
     env.filters["humanize"] = humanize_enum
     env.filters["strip_md"] = strip_markdown
     ```
