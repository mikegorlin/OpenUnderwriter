   - Recovery at index 6 (100 >= 100) -> 4 trading days

   - Prices: [100, 90, 85, 82, 80]

   - period_days should be sum of individual periods

4. `test_group_non_consecutive_drops`:
   - Create 2 StockDropEvent objects separated by 5 days
   - After grouping, should produce 2 separate events (not merged)

5. `test_tag_market_wide_events`:

   - SPY history with -4% on same date -> is_market_wide = True

6. `test_tag_company_specific_event`:
   - Create a StockDropEvent on "2025-03-15"
   - SPY history with +0.5% on same date -> is_market_wide = False

Build synthetic history dicts matching the column-oriented format: `{"Date": [...], "Close": [...]}`.
