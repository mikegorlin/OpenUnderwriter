       - `font_heading = "Georgia"` (serif for headings -- close to Perpetua, widely available)

       - `page_margin = Inches(0.75)`
       - `chart_width = Inches(6.5)` (full width within margins for letter-size)
       - `chart_dpi = 200`

     - "DOCitation" with font_mono, color_text_light, small size
   - `get_risk_color(level: str) -> str` function: maps risk level strings (CRITICAL, HIGH, ELEVATED, MODERATE, LOW, NEUTRAL) to hex color strings

   - `set_cell_shading(cell, hex_color: str) -> None`: Apply background color to a table cell via OxmlElement. Create a NEW element each time (do NOT reuse across cells).
   - `add_styled_table(doc: Document, headers: list[str], rows: list[list[str]], ds: DesignSystem) -> Table`: Create a table with navy header row (white text, bold), alternating row shading, and consistent cell padding.
   - `add_data_table(doc: Document, headers: list[str], rows: list[list[str]], ds: DesignSystem, col_widths: list[float] | None = None) -> Table`: Like add_styled_table but with optional column width control and right-aligned numeric columns.
   - `add_risk_indicator(paragraph, level: str, ds: DesignSystem) -> None`: Add a colored risk level text (e.g., "[HIGH]") using the risk heat spectrum colors.
   - `add_page_number(paragraph) -> None`: Add auto-updating page number field using OxmlElement fldChar + instrText with PAGE instruction.
   - `add_toc_field(doc: Document) -> None`: Insert a TOC field code that Word will regenerate on open. Uses OxmlElement with TOC instruction.
   - `set_cell_border(cell, **kwargs) -> None`: Set individual cell borders (top, bottom, left, right) with color and width.
   - `add_sourced_paragraph(doc: Document, text: str, citation: str, ds: DesignSystem) -> Paragraph`: Add a paragraph with main text and a smaller, lighter citation suffix.
   - `set_table_column_widths(table: Table, widths: list[Inches]) -> None`: Set column widths on a table.

   - `embed_chart(doc: Document, buf: io.BytesIO, width: Inches | None = None) -> None`: Add a picture from BytesIO to the document at specified width.
   - `create_figure(width: float = 6.5, height: float = 3.5, dpi: int = 200) -> tuple[Figure, Axes]`: Create a new figure with consistent defaults and the Agg backend.
   - `apply_chart_style(ax: Axes, ds: DesignSystem, title: str = "", ylabel: str = "") -> None`: Apply consistent chart styling: navy title, gray grid, white background, spine cleanup.

Uses polar axes, navy fill with alpha, gold accent for the outline, category labels around the perimeter. Returns BytesIO ready for embedding.
