2. **Self-review checklist** -- evaluate the document against these quality dimensions:

   **Typography & Hierarchy:**
   - [ ] Heading sizes create clear visual hierarchy (H1 > H2 > H3)
   - [ ] Body text is readable at 10pt or similar
   - [ ] Line spacing is comfortable (not cramped, not wasteful)
   - [ ] Font pairing works (serif headings + sans-serif body)
   - [ ] Data numbers are right-aligned, text is left-aligned
   - [ ] Citations are visibly smaller/lighter than main text (superscript or smaller font)

   **Color & Branding:**
   - [ ] Navy (#1A1446) is the dominant professional color
   - [ ] Gold (#FFD000) accents are used sparingly (headers, dividers, key callouts)
   - [ ] Risk colors are immediately readable (red = bad, blue = not bad)
   - [ ] No green anywhere in risk indicators
   - [ ] Table headers are navy with white text
   - [ ] Alternating row shading is subtle

   **Charts:**
   - [ ] Radar chart is readable with all 10 factor labels visible
   - [ ] Stock chart annotations don't overlap
   - [ ] Ownership donut chart has clear labels
   - [ ] Timeline chart events don't clutter
   - [ ] Chart titles are informative
   - [ ] Chart sizing fills the page width nicely

   **Tables:**
   - [ ] Column widths are appropriate (not too wide, not truncating content)
   - [ ] Financial numbers are properly formatted and aligned
   - [ ] Conditional formatting colors are visible but not overwhelming
   - [ ] Header row is clearly distinct from data rows

   **Layout & Flow:**
   - [ ] TOC appears after title
   - [ ] Executive summary is the first real content
   - [ ] Continuous flow between sections (no awkward page breaks)
   - [ ] Footer shows page numbers
   - [ ] Header shows company name

   **Data Presentation:**
   - [ ] Narrative paragraphs lead each section (story-forward)
   - [ ] Source citations are present but not distracting
   - [ ] Key findings stand out visually
   - [ ] Tier/score display is prominent but contextual
   - [ ] Risk indicators are color-coded consistently

   - Polish heading styles, paragraph spacing, and page layout
   - Ensure the executive summary page "tells the story" at a glance
