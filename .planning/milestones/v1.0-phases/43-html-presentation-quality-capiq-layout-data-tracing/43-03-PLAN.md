    **Part A — formatters.py: Add format_em_dash function**

    Append this function to `formatters.py` (do NOT modify existing `format_na` — backward compatibility required):

    ```python
    def format_em_dash(value: object) -> str:
        """Return em dash for None/empty values, otherwise str(value).

        Used in 3-column data grid for CapIQ-style missing data display.
        Leaves existing format_na('N/A') calls unchanged for backward compatibility.
        """
        if value is None or value == "" or value == []:
            return "\u2014"  # em dash
        return str(value)
    ```

    Append to `components/tables.html.j2` (currently 171 lines — will stay under 500):

    ```jinja2
    {# ============================================================
       3-column data grid macros — CapIQ-grade density display.
       Label | Value | Context/Benchmark

       data_row: renders a single 3-column row.
         label:        metric name (string)
         value:        formatted value; if falsy, em dash is shown
         context_text: peer benchmark, threshold, or context string.
                       Empty string if no context — column shows blank (not "N/A")
         footnote_num: integer footnote number, or 0/none for no footnote
         confidence:   "HIGH" (silent), "MEDIUM" -> (est.), "LOW" -> (web)

       data_grid: wraps data_row calls in a structured table.
         title:        optional bold header row spanning all 3 columns
         Use with {% call(row) data_grid("Title") %}...{% endcall %}
    ============================================================ #}

    {% macro data_row(label, value="", context_text="", footnote_num=0, confidence="HIGH") %}
    <tr class="dr-row">
      <td class="dr-label">{{ label }}</td>
      <td class="dr-value">
        {%- if value is not none and value != "" -%}
          {{ value }}
        {%- else -%}
          <span class="dr-missing">&mdash;</span>
        {%- endif -%}
        {%- if footnote_num %}<sup class="fn-ref"><a href="#fn-{{ footnote_num }}">{{ footnote_num }}</a></sup>{% endif -%}
        {%- if confidence == "MEDIUM" %}<span class="conf-marker"> (est.)</span>{% endif -%}
        {%- if confidence == "LOW" %}<span class="conf-marker"> (web)</span>{% endif -%}
      </td>
      <td class="dr-context">{{ context_text }}</td>
    </tr>
    {% endmacro %}

    {% macro data_grid(title="") %}
    <table class="w-full border-collapse text-sm data-grid-table">
      {% if title %}
      <thead>
        <tr><th colspan="3" class="data-grid-title text-left">{{ title }}</th></tr>
      </thead>
      {% endif %}
      <tbody>
        {{ caller() }}
      </tbody>
    </table>
    {% endmacro %}
    ```
