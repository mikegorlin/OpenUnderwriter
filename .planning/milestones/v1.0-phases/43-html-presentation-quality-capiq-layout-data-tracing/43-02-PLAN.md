    **Create `src/do_uw/templates/html/sections/red_flags.html.j2`:**

    Pulls red_flags data from the scoring context (pre-built in Python before render). The `scoring` variable contains `red_flags` as a list of flag dicts. Priority-sorted: worst severity first.

    Severity ordering: TRIGGERED > ELEVATED > INFO > CLEAR. Sort descending.

    ```jinja2
    {# Red Flags — dedicated section, immediately after Executive Summary.
       Priority-sorted table: Severity | Check name | Finding | Source.
       Only shows TRIGGERED and ELEVATED flags (not CLEAR/INFO).
       Section id="red-flags" for sidebar TOC anchor. #}
    {% set sc = scoring or {} %}
    {% set all_flags = sc.get('red_flags', []) %}

    {# Filter to triggered/elevated only, sort by severity weight descending #}
    {% set severity_weight = {'TRIGGERED': 3, 'ELEVATED': 2, 'INFO': 1, 'CLEAR': 0} %}
    {% set active_flags = [] %}
    {% for flag in all_flags %}
      {% set status = flag.get('status', flag.get('ceiling', '')) | upper %}
      {% if status in ['TRIGGERED', 'ELEVATED'] %}
        {% set _ = active_flags.append(flag) %}
      {% endif %}
    {% endfor %}
    {# Sort by ceiling/status descending — TRIGGERED before ELEVATED #}
    {% set sorted_flags = active_flags | sort(attribute='ceiling', reverse=True) if active_flags else [] %}

    <section id="red-flags" class="page-break">
      <h2>Red Flags</h2>
      {% if sorted_flags %}
      <p class="text-sm text-gray-600 mb-2">
        <strong>{{ sorted_flags | length }}</strong> risk signal{{ 's' if (sorted_flags | length) != 1 else '' }} require underwriter attention.
        Reviewed in severity order — critical issues first.
      </p>
      <table class="w-full border-collapse text-sm my-3">
        <thead>
          <tr class="bg-navy text-white">
            <th class="text-left px-3 py-2 w-24">Severity</th>
            <th class="text-left px-3 py-2 w-48">Flag</th>
            <th class="text-left px-3 py-2">Finding</th>
            <th class="text-left px-3 py-2 w-32">Source</th>
          </tr>
        </thead>
        <tbody>
          {% for flag in sorted_flags %}
          {% set status = flag.get('status', flag.get('ceiling', 'TRIGGERED')) | upper %}
          <tr class="{{ 'bg-red-50' if status == 'TRIGGERED' else 'bg-amber-50' }}">
            <td class="px-3 py-2">{{ traffic_light(status, status) }}</td>
            <td class="px-3 py-2 font-semibold text-xs">{{ flag.get('name', flag.get('check_id', '—')) }}</td>
            <td class="px-3 py-2 text-xs">{{ flag.get('description', flag.get('finding', '—')) }}</td>
            <td class="px-3 py-2 text-xs text-gray-500">
              {% set ev = flag.get('evidence', []) %}
              {{ ev[0] if ev else (flag.get('source', '—')) }}
            </td>
          </tr>
          {% endfor %}
        </tbody>
      </table>
      {% else %}
      <p class="text-sm text-gray-500 my-3">No triggered or elevated risk signals identified.</p>
      {% endif %}
    </section>
    ```
