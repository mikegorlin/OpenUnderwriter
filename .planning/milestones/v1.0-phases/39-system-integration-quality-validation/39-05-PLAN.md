  <action>
1. **Add pipeline_metadata field to AnalysisState** (in `src/do_uw/models/state.py`):
   ```python
   pipeline_metadata: dict[str, Any] = Field(default_factory=dict)
   ```
   This stores per-run metadata like token counts, costs, timestamps, and data freshness. Using a dict (not a typed model) because metadata content varies by run configuration.

2. **Wire CostTracker summary into state at ExtractStage completion** (in `src/do_uw/stages/extract/__init__.py`):
   - At the end of `ExtractStage.run()`, after all extractions complete:
   - Check if LLMExtractor was used (it may not be in test/cache-only runs)
   - If yes, call `extractor.cost_summary()` and store result in `state.pipeline_metadata["llm_cost"]`
   - Also store `state.pipeline_metadata["data_freshness_date"]` = today's ISO date
   - Also store `state.pipeline_metadata["extraction_timestamp"]` = current ISO datetime

3. **Add pipeline-level cost logging** (in `src/do_uw/pipeline.py`):
   - After each stage completes, if `state.pipeline_metadata.get("llm_cost")` exists, log a summary line:
     ```
     [EXTRACT] LLM tokens: {input}→{output}, estimated cost: ${cost:.4f}
     ```
   - At pipeline completion, log total metadata summary

4. **Ensure cost_summary() returns complete data** (in `src/do_uw/stages/extract/llm/extractor.py`):
   - Verify `LLMExtractor.cost_summary()` returns: `{"input_tokens": int, "output_tokens": int, "total_cost_usd": float, "extractions": int, "budget_usd": float}`
   - If it doesn't include all fields, add them

5. Handle the case where pipeline runs from cached state (no LLM extraction): `pipeline_metadata["llm_cost"]` should default to `{"input_tokens": 0, "output_tokens": 0, "total_cost_usd": 0.0, "extractions": 0}` or simply not be set.
