    **New file: `pricing_inference.py`** — Rate inference engine:

    1. `load_rate_decay_config() -> dict[str, float]` — Load config/rate_decay.json. Use Path(__file__).parent.parent.parent / "config" / "rate_decay.json" to find it. Return dict of factor names to float values. Fall back to hardcoded defaults if file not found (defensive).

    2. `infer_layer_rate(primary_rate: float, layer_number: int, layer_type: str, decay_config: dict[str, float] | None = None) -> tuple[float, str]` — Per D1/D7 decision:
       - If layer_type == "PRIMARY": return (primary_rate, "PRIMARY: no inference needed")
       - If layer_type == "SIDE_A": factor = decay_config.get("side_a_factor", 0.25); return (primary_rate * factor, "INFERRED: Side A at {factor:.0%} of primary")
       - If layer_type == "EXCESS": key = f"excess_{layer_number}"; factor = decay_config.get(key, max(0.10, 0.60 - 0.08 * layer_number)); return (primary_rate * factor, f"INFERRED: Excess {layer_number} at {factor:.0%} of primary")
       - Always loads decay_config from file if not provided

    3. `infer_primary_from_excess(excess_rate: float, layer_number: int, decay_config: dict[str, float] | None = None) -> tuple[float, str]` — Back-calculate primary rate from known excess layer:
       - factor = decay_config[f"excess_{layer_number}"] or default
       - primary = excess_rate / factor
       - Return (primary, f"INFERRED: Primary back-calculated from excess {layer_number} at {factor:.0%}")

    4. `infer_missing_layers(known_layers: list[dict[str, Any]], total_layers: int | None = None) -> list[dict[str, Any]]` — Main inference function:
       - Input: list of layer dicts with keys: layer_type, layer_number, rate_on_line (float | None), premium (float | None), limit_amount (float | None)
       - Find primary rate: from known primary layer, or back-calculate from lowest known excess
       - If primary rate found: for each layer missing rate_on_line, infer it using infer_layer_rate
       - If premium is known but rate is not (and limit is known): compute rate = premium/limit, mark VERIFIED
       - If rate is inferred and limit is known: compute premium = rate * limit, mark INFERRED
       - Return all layers (original + inferred) with data_source field set appropriately
       - If no primary and no excess layers have rates: return original layers unchanged (can't infer from nothing)

    5. `compute_data_completeness(layers: list[dict[str, Any]]) -> str` — Determine DataCompleteness:
       - If all layers have verified premium+limit: "COMPLETE"
       - If some layers have data: "PARTIAL"
       - If fewer than 2 data points: "FRAGMENT"

    **New file: `pricing_ingestion.py`** — Document ingestion pipeline:

    1. **Pydantic models for LLM extraction** (at top of file):
       ```python
       class ExtractedLayer(BaseModel):
           """Layer extracted from a document by LLM."""
           layer_type: str = Field(description="PRIMARY, EXCESS, or SIDE_A")
           layer_label: str | None = Field(default=None, description="e.g. 'Primary', '1st Excess'")
           layer_number: int = Field(description="Ordinal layer number")
           limit: float | None = Field(default=None, description="Layer limit in USD")
           premium: float | None = Field(default=None, description="Layer premium in USD")
           carrier: str | None = Field(default=None, description="Insurance carrier name")
           attachment: float | None = Field(default=None, description="Attachment point in USD")
           retention: float | None = Field(default=None, description="SIR/deductible (primary only)")
           commission_pct: float | None = Field(default=None)
           am_best_rating: str | None = Field(default=None)

       class ExtractedProgram(BaseModel):
           """Structured pricing data extracted from a document."""
           company_name: str | None = Field(default=None)
           ticker: str | None = Field(default=None)
           effective_date: str | None = Field(default=None, description="YYYY-MM-DD format")
           expiration_date: str | None = Field(default=None)
           policy_year: int | None = Field(default=None)
           total_limit: float | None = Field(default=None)
           total_premium: float | None = Field(default=None)
           retention: float | None = Field(default=None)
           layers: list[ExtractedLayer] = Field(default_factory=list)
           broker_name: str | None = Field(default=None)
           brokerage: str | None = Field(default=None)
       ```

    2. `extract_text_from_excel(filepath: Path) -> str` — Convert Excel to text for LLM:
       - Use openpyxl `load_workbook(filepath, read_only=True, data_only=True)`
       - For each sheet: append "=== Sheet: {name} ===" header
       - For each row: join cell values with tab separator, skip empty rows
       - Return full text representation
       - Handles .xlsx and .xls (openpyxl handles .xlsx; for .xls, catch exception and suggest conversion)

    3. `extract_text_from_pdf(filepath: Path) -> str` — Convert PDF to text:
       - Use pdfplumber: `with pdfplumber.open(filepath) as pdf:`
       - For each page: extract text, extract tables (format as tab-separated)
       - Combine page text + table text
       - Return full text

    4. `extract_text_from_file(filepath: Path) -> str` — Dispatcher:
       - .xlsx / .xls -> extract_text_from_excel
       - .pdf -> extract_text_from_pdf
       - .txt / .md / .csv -> read file directly
       - Other: raise ValueError with supported formats list

    5. `extract_pricing_from_text(text: str, hint: str | None = None) -> ExtractedProgram` — LLM extraction:
       - Uses instructor library with Anthropic client
       - System prompt explaining D&O tower semantics (from D1): primary has retention, excess layers stack, Side A sits on top, "10xs10" notation, etc.
       - If hint provided, add context: "This document is a {hint}"
       - Creates instructor client: `instructor.from_provider("anthropic/claude-sonnet-4-20250514")` (use Sonnet for cost/speed — haiku may not be available)
       - Calls client.create with response_model=ExtractedProgram
       - Returns ExtractedProgram
       - Requires ANTHROPIC_API_KEY env var. If missing, raise RuntimeError with helpful message.

    6. `ingest_document(filepath: Path, ticker: str, store: Any, hint: str | None = None) -> dict[str, Any]` — End-to-end pipeline:
       - Extract text from file
       - Extract pricing via LLM
       - Map ExtractedProgram to ProgramInput + PolicyYearInput + EnhancedLayerInput(s)
       - Override ticker with provided ticker argument (user is the authority)
       - Set data_source=DataSource.AI_EXTRACTED on all layers
       - Set source="ai_extracted" and source_document=filepath.name on PolicyYear
       - Use ProgramStore.get_program_by_ticker to find/create program
       - Use ProgramStore.add_policy_year to store
       - Return summary dict: {"program_id": int, "policy_year_id": int, "layers_extracted": int, "data_completeness": str}

    - The LLM call is sync (project uses sync patterns per memory)
