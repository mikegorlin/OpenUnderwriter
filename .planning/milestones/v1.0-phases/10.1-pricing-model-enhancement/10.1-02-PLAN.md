       - appetite_notes (String, nullable)
       - notes_text (String, nullable)
       - created_at (DateTime, NOT NULL)

    3. `_create_programs_table()` — Create programs table (FK to brokers):
       - id (Integer, PK, autoincrement)
       - ticker (String, NOT NULL)
       - company_name (String, nullable)
       - anniversary_month (Integer, nullable)
       - anniversary_day (Integer, nullable)
       - broker_id (Integer, nullable) — FK added via batch_alter_table
       - notes_text (String, nullable)
       - created_at (DateTime, NOT NULL)
       - updated_at (DateTime, NOT NULL)
       - After creation, use batch_alter_table to add FK constraint to brokers.id

    4. `_create_policy_years_table()` — Create policy_years table (FK to programs):
       - id (Integer, PK, autoincrement)
       - program_id (Integer, NOT NULL) — FK added via batch
       - policy_year (Integer, NOT NULL)
       - effective_date (DateTime, nullable)
       - expiration_date (DateTime, nullable)
       - total_limit (Float(), nullable)
       - total_premium (Float(), nullable)
       - retention (Float(), nullable)
       - status (String, NOT NULL, server_default="QUOTED")
       - data_completeness (String, NOT NULL, server_default="FRAGMENT")
       - source (String, NOT NULL, server_default="manual")
       - source_document (String, nullable)
       - program_rate_on_line (Float(), nullable)
       - notes_text (String, nullable)
       - created_at (DateTime, NOT NULL)
       - After creation, use batch_alter_table to add FK to programs.id

    5. `_modify_tower_layers()` — Add new columns to existing tower_layers via batch_alter_table:
       - layer_type (String, nullable) — new
       - layer_label (String, nullable) — new
       - commission_pct (Float(), nullable) — new
       - data_source (String, nullable) — new
       - carrier_id (Integer, nullable) — new FK to carriers.id
       - policy_year_id (Integer, nullable) — new FK to policy_years.id
       - Add FK constraints for carrier_id -> carriers.id and policy_year_id -> policy_years.id

    6. `_create_indexes()` — Indexes for common query patterns:
       - ix_programs_ticker on programs(ticker)
       - ix_policy_years_program_id on policy_years(program_id)
       - ix_policy_years_policy_year on policy_years(policy_year)
       - ix_carriers_carrier_name on carriers(carrier_name)
       - ix_brokers_brokerage_name on brokers(brokerage_name)
       - ix_tower_layers_policy_year_id on tower_layers(policy_year_id)
       - ix_tower_layers_carrier_id on tower_layers(carrier_id)
       - ix_tower_layers_layer_type on tower_layers(layer_type)

    7. `_migrate_existing_quotes()` — Data migration (DML within migration):

       - For each quote: INSERT INTO policy_years (program_id, policy_year, effective_date, ...) mapping from the quote fields
       - Update tower_layers.policy_year_id to point to the new policy_year records
       - Use raw SQL via op.execute() for the data migration (Alembic standard for DML)
       - Use `sa.text()` for SQL strings (not raw strings) for SQLAlchemy 2.0 compliance

    **New file: `pricing_store_programs.py`** — Program and PolicyYear CRUD:

    Create `ProgramStore` class following PricingStore patterns (contextmanager _session, engine from db_path, lazy table creation):

    Methods:

    1. `add_program(input: ProgramInput) -> int` — Create a Program. If input.broker is provided (BrokerInput), create or find broker first. If input.broker_id is provided, use that. Auto-set created_at/updated_at. Return program ID.

    2. `get_program(program_id: int) -> ProgramOutput | None` — Get program with eager-loaded policy_years and broker. Return ProgramOutput Pydantic model.

    3. `get_program_by_ticker(ticker: str) -> ProgramOutput | None` — Find program by ticker (most recent if multiple). Eager-load policy_years.

    4. `list_programs(ticker: str | None = None, limit: int = 50) -> list[ProgramOutput]` — List programs, optionally filtered by ticker.

    5. `add_policy_year(program_id: int, input: PolicyYearInput) -> int` — Add a policy year to a program. Compute program_rate_on_line if total_premium and total_limit are both provided (use _safe_divide from pricing_store). Create TowerLayer records from input.layers (compute rate_on_line, premium_per_million where premium and limit_amount are both provided; set to None/0.0 when data is missing). Set layer_type, layer_label, data_source from EnhancedLayerInput. Return policy_year ID. Raise ValueError if program_id not found.

    6. `get_policy_year(policy_year_id: int) -> PolicyYearOutput | None` — Get with eager-loaded layers.

    7. `get_program_history(ticker: str) -> list[PolicyYearOutput]` — Get all policy years for a ticker's program, ordered by policy_year desc. This enables YoY tracking (D8).

    8. `add_broker(input: BrokerInput) -> int` — Add a broker record. Return ID.

    9. `get_or_create_broker(brokerage_name: str, producer_name: str | None = None) -> int` — Find by brokerage_name (case-insensitive), create if not exists. Return ID.

    10. `add_carrier(input: CarrierInput) -> int` — Add a carrier record. Return ID.

    11. `get_or_create_carrier(carrier_name: str) -> int` — Find by name, create if not exists. Return ID.

    12. `list_brokers(brokerage_name: str | None = None) -> list[BrokerOutput]` — List brokers, optionally filtered.
