    **New file: `cli_pricing_programs.py`** — Program management CLI commands:

    Create `pricing_programs_app = typer.Typer(name="programs", help="D&O program management", no_args_is_help=True)`.

    Commands (all with lazy imports inside functions):

    1. `add-program` — Create a new D&O program:
       - Arguments: `ticker` (required)
       - Options: `--company-name`, `--anniversary-month` (int), `--anniversary-day` (int), `--brokerage` (str), `--broker-name` (str), `--broker-email`, `--broker-phone`, `--notes`
       - Creates ProgramInput, optionally with BrokerInput if brokerage provided
       - Calls ProgramStore.add_program
       - Prints: "Program #{id} created for {TICKER}"

    2. `list-programs` — List D&O programs:
       - Options: `--ticker` filter, `--limit` (default 20)
       - Rich table: ID, Ticker, Company, Anniversary, Broker, Policy Years
       - Anniversary displayed as MM/DD if known, "-" if not

    3. `program-history` — Show year-over-year program evolution (D8):
       - Arguments: `ticker` (required)
       - Calls ProgramStore.get_program_history(ticker)
       - Rich table: Year, Effective, Status, Limit, Premium, ROL, Retention, Completeness, Layers
       - Below table: show YoY changes using compute_yoy_changes
       - Show carrier rotations using detect_carrier_rotations

    4. `add-policy-year` — Add a policy year to a program:
       - Arguments: `ticker` (required), `year` (int, required)
       - Options: `--premium`, `--limit`, `--retention`, `--status`, `--effective` (date)
       - Finds program by ticker (or creates if not exists)
       - Creates PolicyYearInput and calls ProgramStore.add_policy_year
       - Prints: "Policy year {year} added to {TICKER} program"

    5. `brokers` — List brokers in the database:
       - Options: `--brokerage` filter
       - Rich table: ID, Brokerage, Producer, Email, Phone

    6. `ingest` — Ingest a document into pricing database (D6):
       - Arguments: `filepath` (Path), `ticker` (required)
       - Options: `--hint` (str, e.g. "tower spreadsheet", "program summary")
       - Lazy imports pricing_ingestion.ingest_document
       - Calls ingest_document(filepath, ticker, store, hint)
       - Prints: summary of extracted data (layers, completeness)
       - On error (no API key, bad file): print helpful error message
