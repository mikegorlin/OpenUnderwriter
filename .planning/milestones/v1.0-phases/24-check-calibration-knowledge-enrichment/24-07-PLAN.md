**cli_calibrate.py:**
Ensure the `calibrate run` command orchestrates the FULL pipeline:
1. Run CalibrationRunner on 12 tickers (or --no-fresh to use cached output)
2. After all tickers complete: run CheckAnalyzer on the report
3. Run ImpactRanker to get top N checks
4. Run AnomalyDetector to find anomalies
5. Run ReportGenerator to produce summary + per-ticker Markdown
6. Save all artifacts to output/calibration/
7. Print Rich summary to console

Add a `calibrate all` convenience command that runs: run -> report -> playbooks -> fix (dry-run) -> enrich in sequence. This is the "one command does everything" entry point.

Options for `calibrate all`: `--output`, `--fresh/--no-fresh`, `--no-llm`, `--top-n`, `--auto-promote`, `--apply-fixes/--dry-run-fixes`

Ensure each individual command (`run`, `report`, `playbooks`, `fix`, `enrich`) can also be run independently for selective re-execution.

   ```python

   ```

3. Assert:

This is NOT a pipeline execution test (too expensive). It validates that all modules compose correctly with synthetic data.
