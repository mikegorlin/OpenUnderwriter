### Add peril sections to HTML template

Insert after Tier Classification, before 10-Factor Scoring in `scoring.html.j2`:

```html
{# --- D&O Peril Assessment (Phase 42) --- #}
{% set ps = sc.get('peril_scoring', {}) %}
{% if ps and ps.get('all_perils') %}
<h3>D&O Claim Peril Assessment</h3>
<p class="text-sm text-gray-600 mb-2">
  {{ ps.active_count }} of 8 D&amp;O claim perils show active risk signals.
</p>
<table class="w-full border-collapse my-3 text-sm">
  <thead>
    <tr class="bg-navy text-white">
      <th class="...">Peril</th>
      <th class="...">Risk Level</th>
      <th class="...">Active Chains</th>
      <th class="...">Key Evidence</th>
    </tr>
  </thead>
  <tbody>
    {% for peril in ps.all_perils %}
    <tr>
      <td class="...font-semibold">{{ peril.name }}</td>
      <td>{{ traffic_light(peril.risk_level, peril.risk_level|humanize) }}</td>
      <td>{{ peril.active_chain_count }}/{{ peril.total_chain_count }}</td>
      <td class="text-xs text-gray-600">{{ peril.key_evidence|join('; ')|truncate(60) }}</td>
    </tr>
    {% endfor %}
  </tbody>
</table>

{# Per-peril deep dives #}
{% for peril in ps.get('perils', []) %}
<div class="border-l-4 {{ 'border-red-700' if peril.risk_level == 'HIGH' else 'border-amber-500' if peril.risk_level == 'ELEVATED' else 'border-blue-400' }} bg-bg-alt pl-4 py-3 my-3 page-break-inside-avoid">
  <h4 class="text-sm font-bold m-0">{{ peril.name }}: {{ peril.risk_level }}</h4>
  <p class="text-xs text-gray-500 m-0 mt-1">
    Frequency: {{ peril.frequency }} | Severity: {{ peril.severity }}
    {% if peril.typical_settlement_range %} | Range: {{ peril.typical_settlement_range }}{% endif %}
  </p>
  {% for chain in peril.chains if chain.active %}
  <div class="mt-2 ml-2">
    <p class="text-sm font-semibold m-0">{{ chain.name }}</p>
    {% if chain.triggered_triggers %}
    <p class="text-xs m-0 mt-1"><span class="text-red-700 font-semibold">Triggers:</span>
      {% for t in chain.triggered_triggers %}{{ t }}{% if not loop.last %}, {% endif %}{% endfor %}</p>
    {% endif %}
    {% if chain.active_amplifiers %}
    <p class="text-xs m-0"><span class="text-amber-700 font-semibold">Amplifiers:</span>
      {% for a in chain.active_amplifiers %}{{ a }}{% if not loop.last %}, {% endif %}{% endfor %}</p>
    {% endif %}
    {% if chain.active_mitigators %}
    <p class="text-xs m-0"><span class="text-green-700 font-semibold">Mitigators:</span>
      {% for m in chain.active_mitigators %}{{ m }}{% if not loop.last %}, {% endif %}{% endfor %}</p>
    {% endif %}
  </div>
  {% endfor %}
</div>
{% endfor %}
{% endif %}
```
