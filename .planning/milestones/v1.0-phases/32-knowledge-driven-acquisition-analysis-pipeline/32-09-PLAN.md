    **Create `src/do_uw/stages/extract/brain_hints.py` (~130 lines):**

    1. `ExtractionHint` — a small TypedDict or Pydantic model:
       ```python
       class ExtractionHint(TypedDict, total=False):
           field_patterns: list[str]
           expected_type: str
           no_data_sentinel: str
           validation_rule: str
       ```

    2. `ExtractionHintsMap` type alias: `dict[str, ExtractionHint]` (check_id -> hint)

    3. `load_extraction_hints() -> ExtractionHintsMap`:
       - Import BrainDBLoader lazily
       - Load checks, filter to those with extraction_hints
       - Return dict mapping check_id -> extraction_hints dict
       - Graceful fallback: return empty dict if brain unavailable
       - Log summary: "Loaded extraction hints for N checks from brain"

    4. `get_hints_for_source(hints: ExtractionHintsMap, source_type: str, checks: list[dict]) -> dict[str, ExtractionHint]`:
       - Given a source type (e.g., "SEC_10K"), return only the hints for checks that require that source
       - Used by individual extractors to know what the brain expects them to find

    5. `validate_extraction_against_hints(hints: ExtractionHintsMap, extracted_fields: set[str]) -> dict`:
       - Compare what brain expected (from hints) vs. what was actually extracted
       - Return: {"satisfied": [...], "missing": [...], "coverage_pct": float}
       - Log WARNING for missing fields that have hints

    **Modify `src/do_uw/stages/extract/__init__.py`:**

    6. In `ExtractStage.__init__()`, add `self._extraction_hints: ExtractionHintsMap = {}` field

    7. In `ExtractStage.run()`, at the VERY START (before Phase 0):
       ```python
       try:
           from do_uw.stages.extract.brain_hints import load_extraction_hints
           self._extraction_hints = load_extraction_hints()
           if self._extraction_hints:
               logger.info("Brain extraction hints loaded for %d checks", len(self._extraction_hints))
       except Exception:
           logger.warning("Failed to load brain extraction hints; continuing without hints")
           self._extraction_hints = {}
       ```

    8. At the END of ExtractStage.run() (before mark_stage_completed), add hint validation:
       ```python
       if self._extraction_hints:
           from do_uw.stages.extract.brain_hints import validate_extraction_against_hints
           # Collect all field keys that were successfully extracted
           extracted_keys = _collect_extracted_field_keys(state)
           validation = validate_extraction_against_hints(self._extraction_hints, extracted_keys)
           logger.info("Brain hint coverage: %.1f%% (%d/%d hint-declared fields extracted)",
                       validation["coverage_pct"], len(validation["satisfied"]),
                       len(validation["satisfied"]) + len(validation["missing"]))
       ```
