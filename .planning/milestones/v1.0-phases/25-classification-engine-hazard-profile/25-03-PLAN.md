  <action>
    Create `src/do_uw/stages/hazard/hazard_engine.py`:
    - `compute_hazard_profile(extracted: ExtractedData, company: CompanyProfile, classification: ClassificationResult, weights_config: dict[str, Any], interactions_config: dict[str, Any]) -> HazardProfile`
    - Steps:
      1. Call `score_all_dimensions()` from dimension_scoring to get list of HazardDimensionScore
      2. `aggregate_by_category()`: Group dimension scores by category. For each category: raw_score = sum of (normalized_score * dimension_weight_within_category) / count, weighted_score = raw_score * category_weight_pct / 100. Track data_coverage_pct per category.
      3. Compute raw_ies = sum of all category weighted_scores. This gives a 0-100 base IES.
      4. Call `detect_named_interactions()` and `detect_dynamic_interactions()` from interaction_effects
      5. `compute_interaction_multiplier()`: Combine named and dynamic interaction multipliers. Named interactions apply their multiplier (interpolated within multiplier_range based on how far the triggering dimensions exceed thresholds). Dynamic interactions apply their modest multiplier. Total interaction multiplier = product of all individual multipliers, capped at 2.0x (to prevent explosion per Pitfall 3 in research).
      6. adjusted_ies = min(100.0, raw_ies * interaction_multiplier)
      7. `_ies_to_filing_multiplier(ies, breakpoints)`: Piecewise linear interpolation from config breakpoints. IES=50 -> 1.0x per user decision.
      8. Compute data_coverage_pct = (dimensions with data_available=True) / 47 * 100
      9. If data_coverage_pct < 60%, set confidence_note to low coverage warning
      10. Collect underwriter_flags from dimension evidence notes that start with "MEETING_PREP:"
    - `load_hazard_config() -> tuple[dict, dict]`: Load hazard_weights.json and hazard_interactions.json from config/ directory

    Create `src/do_uw/stages/hazard/interaction_effects.py`:
    - `detect_named_interactions(dimension_scores: list[HazardDimensionScore], config: dict[str, Any]) -> list[InteractionEffect]`
      - For each named interaction in config, check if ALL required dimensions meet their min_score_pct threshold
      - When triggered, interpolate multiplier within multiplier_range based on average excess above threshold
      - Return InteractionEffect with is_named=True
    - `detect_dynamic_interactions(dimension_scores: list[HazardDimensionScore], config: dict[str, Any]) -> list[InteractionEffect]`
      - Count dimensions scoring above elevated_threshold_pct (default 60%)
      - If count >= min_elevated_dimensions (default 5): create "Elevated Co-occurrence" effect
      - Interpolate multiplier within co_occurrence_multiplier range based on count above threshold
      - Category concentration check: if any category has 3+ elevated dimensions, create category-specific effect
      - Return InteractionEffect with is_named=False

    Update `src/do_uw/stages/hazard/__init__.py`:
    - Add re-export of compute_hazard_profile from hazard_engine
    - Add re-export of detect_named_interactions, detect_dynamic_interactions from interaction_effects
