Create `src/do_uw/models/ai_risk.py` with these Pydantic v2 models (pyright strict, `from __future__ import annotations`):

1. **AISubDimension(BaseModel)** -- A single AI risk sub-dimension score:
   - `dimension: str` -- one of: "revenue_displacement", "cost_structure", "competitive_moat", "workforce_automation", "regulatory_ip"
   - `score: float` -- 0-10 scale
   - `weight: float` -- Industry-specific weight (0.0-1.0, sums to 1.0 across dimensions)
   - `evidence: list[str]` -- Supporting evidence items (default_factory=list)
   - `threat_level: str` -- "HIGH", "MEDIUM", "LOW", "UNKNOWN"

2. **AIDisclosureData(BaseModel)** -- AI disclosure analysis from SEC filings:
   - `mention_count: int = 0` -- Total AI keyword mentions in Item 1A
   - `risk_factors: list[str]` -- Specific AI risk factors disclosed (default_factory=list)
   - `opportunity_mentions: int = 0` -- AI framed as opportunity
   - `threat_mentions: int = 0` -- AI framed as threat
   - `sentiment: str = "UNKNOWN"` -- "OPPORTUNITY", "THREAT", "BALANCED", "UNKNOWN"
   - `yoy_trend: str = "UNKNOWN"` -- "INCREASING", "STABLE", "DECREASING", "UNKNOWN"

3. **AIPatentActivity(BaseModel)** -- Patent filing analysis:
   - `ai_patent_count: int = 0`
   - `recent_filings: list[dict[str, str]]` -- List of {patent_number, filing_date, title} (default_factory=list)
   - `filing_trend: str = "UNKNOWN"` -- "INCREASING", "STABLE", "DECREASING", "UNKNOWN"

4. **AICompetitivePosition(BaseModel)** -- Peer-relative AI positioning:
   - `company_ai_mentions: int = 0`
   - `peer_avg_mentions: float = 0.0`
   - `peer_mention_counts: dict[str, int]` -- Peer ticker -> mention count (default_factory=dict)
   - `percentile_rank: float | None = None` -- 0-100 where 100 = most AI-forward
   - `adoption_stance: str = "UNKNOWN"` -- "LEADING", "INLINE", "LAGGING", "UNKNOWN"

5. **AIRiskAssessment(BaseModel)** -- Complete SECT8 AI risk output:
   - `overall_score: float = 0.0` -- 0-100 weighted composite
   - `sub_dimensions: list[AISubDimension]` (default_factory=list)
   - `disclosure_data: AIDisclosureData` (default_factory=AIDisclosureData)
   - `patent_activity: AIPatentActivity` (default_factory=AIPatentActivity)
   - `competitive_position: AICompetitivePosition` (default_factory=AICompetitivePosition)
   - `industry_model_id: str = "GENERIC"` -- References AI impact model used
   - `disclosure_trend: str = "UNKNOWN"` -- "INCREASING", "STABLE", "DECREASING", "UNKNOWN"
   - `narrative: str = ""` -- Industry-specific risk narrative text
   - `narrative_source: str = ""` -- Source attribution for narrative
   - `narrative_confidence: str = "LOW"` -- Confidence level
   - `peer_comparison_available: bool = False`
   - `forward_indicators: list[str]` (default_factory=list)
   - `data_sources: list[str]` (default_factory=list) -- What sources contributed

**C. Create `src/do_uw/stages/score/ai_risk_scoring.py`:**

Scoring engine function `score_ai_risk(state: AnalysisState) -> AIRiskAssessment`:
1. Load weights from ai_risk_weights.json via `Path(__file__).resolve().parent.parent.parent / "config" / "ai_risk_weights.json"` (3 parents: score/ -> stages/ -> do_uw/ -> config/)
2. Determine industry model using `get_ai_impact_model(sic_code, state.active_playbook_id)`
3. Get industry-specific weights (fall back to "default")
4. Read `state.extracted.ai_risk` (the raw extraction data from EXTRACT stage)
5. Score each sub-dimension (0-10) based on extraction evidence:
   - **revenue_displacement**: Based on AI threat mentions / total mentions ratio, competitor AI announcements, industry threat_level
   - **cost_structure**: Based on workforce_automation indicators, industry cost_structure threat_level
   - **competitive_moat**: Based on patent activity, competitive position (LEADING/LAGGING), industry moat threat_level
   - **workforce_automation**: Based on disclosure mentions of automation, industry workforce threat_level
   - **regulatory_ip**: Based on AI regulatory mentions, patent count, industry regulatory_ip threat_level
6. Compute overall_score = sum(sub_score * weight * 10) for 0-100 range
7. Generate narrative string using industry model data and scores
8. Return populated AIRiskAssessment

Helper functions (public for testability, no underscore prefix):
- `score_revenue_displacement(disclosure: AIDisclosureData, impact_model: dict) -> tuple[float, list[str]]` -- returns (score 0-10, evidence list)
- `score_cost_structure(disclosure: AIDisclosureData, impact_model: dict) -> tuple[float, list[str]]`
- `score_competitive_moat(patents: AIPatentActivity, competitive: AICompetitivePosition, impact_model: dict) -> tuple[float, list[str]]`
- `score_workforce_automation(disclosure: AIDisclosureData, impact_model: dict) -> tuple[float, list[str]]`
- `score_regulatory_ip(disclosure: AIDisclosureData, patents: AIPatentActivity, impact_model: dict) -> tuple[float, list[str]]`
- `generate_ai_risk_narrative(assessment: AIRiskAssessment, industry_name: str) -> str`

Important: The scoring functions should handle empty/default extraction data gracefully (return mid-range scores with "Insufficient data" evidence when no extraction data available). The industry impact model's threat_level acts as a prior: HIGH threat_level starts at 7/10, MEDIUM at 5/10, LOW at 3/10, and extraction evidence adjusts up/down.
