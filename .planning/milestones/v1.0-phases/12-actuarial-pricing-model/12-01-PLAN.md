```python
"""Actuarial expected loss computation.

Implements the frequency-severity expected loss model:
    Expected Loss = E[Filing Probability] x E[Severity | claim] + E[Defense Costs]

Inputs come from existing pipeline stages:
- Filing probability: inherent_risk.py company_adjusted_rate_pct (BENCHMARK)
- Severity scenarios: severity_model.py SeverityScenarios (SCORE)
- Defense cost percentages: actuarial.json config

All parameters are config-driven. No hardcoded values.
"""
```

Functions:
1. `_get_scenario_amount(scenarios: SeverityScenarios, percentile: int) -> float | None` -- extract settlement_estimate at given percentile
2. `_get_defense_cost_pct(case_type: str, actuarial_config: dict[str, Any]) -> float` -- look up from config, default 0.20
3. `compute_expected_loss(filing_probability_pct: float, severity_scenarios: SeverityScenarios | None, case_type: str, actuarial_config: dict[str, Any]) -> ExpectedLoss` -- the main computation

The function:
- Converts filing_probability_pct from percentage to decimal (/ 100.0)
- Gets median severity from 50th percentile scenario
- Returns ExpectedLoss(has_data=False) if no scenarios or no 50th percentile
- Computes expected_indemnity = prob * median_severity
- Computes expected_defense = expected_indemnity * defense_cost_pct
- Computes scenario_losses at each percentile
- Includes methodology_note from config
- Includes assumptions list documenting inputs used
