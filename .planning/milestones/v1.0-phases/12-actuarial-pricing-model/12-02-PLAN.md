**GREEN phase:** Create `src/do_uw/stages/score/actuarial_layer_pricing.py` (~300L):

Module docstring:
```python
"""ILF-based tower layer pricing and market calibration.

Allocates expected loss across D&O tower layers using power curve
ILF(L) = (L/B)^alpha and derives indicated premiums per layer.
Optionally calibrates against market data via credibility weighting.

All parameters from actuarial.json config. No hardcoded values.
"""
```

Functions:

1. `compute_ilf(limit: float, basic_limit: float, alpha: float) -> float`
   - Returns (limit / basic_limit) ** alpha
   - Guard: if basic_limit <= 0 or limit <= 0, return 1.0

2. `_compute_layer_factor(attachment: float, layer_limit: float, basic_limit: float, alpha: float) -> float`
   - ILF(attachment + layer_limit) - ILF(attachment) for excess layers
   - Primary (attachment=0): factor = 1.0 (by definition)
   - Guard: ilf_basic = compute_ilf(basic_limit, basic_limit, alpha) = 1.0

3. `_get_alpha(sector: str | None, market_cap_tier: str | None, actuarial_config: dict[str, Any]) -> float`
   - Lookup from ilf_parameters: try sector-specific key, then cap-specific, then "standard"
   - Map sector codes to config keys: TECH->"standard", FINS->"financial_services", HLTH(biotech)->"biotech", etc.
   - Default: config["ilf_parameters"]["standard"] or 0.40

4. `_layer_type_key(layer_type: str) -> str`
   - Normalize layer_type to loss_ratio_targets key (primary, low_excess, mid_excess, high_excess, side_a)

5. `price_tower_layers(expected_loss: ExpectedLoss, tower_structure: list[LayerSpec], alpha: float, loss_ratio_targets: dict[str, Any], actuarial_config: dict[str, Any]) -> list[LayerPricing]`
   - If not expected_loss.has_data or empty tower: return []
   - Primary layer limit = tower_structure[0].limit (basic limit for ILF)
   - For each layer: compute layer_factor, layer_expected_loss, indicated_premium, ROL
   - Side A layers: use primary expected loss (attachment=0), not excess ILF

6. `calibrate_against_market(model_premium: float, model_rol: float, market_position: MarketPosition, credibility_config: dict[str, Any]) -> CalibratedPricing`
   - If market_position.confidence_level == "INSUFFICIENT": return model-only
   - z = min(1.0, sqrt(n / standard))
   - calibrated_rol = z * market_rol + (1 - z) * model_rol
   - Derive calibrated_premium from calibrated_rol

7. `_load_tower_structure(actuarial_config: dict[str, Any]) -> list[LayerSpec]`
   - Parse default_tower from config into LayerSpec list

8. `build_actuarial_pricing(filing_probability_pct: float, severity_scenarios: SeverityScenarios | None, case_type: str, sector: str | None, market_cap_tier: str | None, market_position: MarketPosition | None, actuarial_config: dict[str, Any]) -> ActuarialPricing`
   - Orchestrates full pipeline: compute_expected_loss -> price_tower_layers -> calibrate_against_market -> ActuarialPricing
   - If no expected loss data: return ActuarialPricing(has_data=False)
   - Computes total_indicated_premium as sum of layer premiums
   - Populates assumptions list with all inputs used
