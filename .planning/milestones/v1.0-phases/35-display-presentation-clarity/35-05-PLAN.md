**Create `src/do_uw/stages/render/html_renderer.py`** (~300 lines):

1. **`build_html_context(state, chart_dir?) -> dict[str, Any]`**
   - Reuse `build_template_context` from `md_renderer.py` as base (same pattern as existing pdf_renderer)
   - Add HTML-specific context:
     - `densities`: state.analysis.section_densities dict (or empty dict if None)
     - `narratives`: state.analysis.pre_computed_narratives (or empty PreComputedNarratives)
     - `chart_images`: base64-encoded chart PNGs (reuse `_load_chart_images` from pdf_renderer)
     - `check_results_by_section`: group check_results by section for check-type display
     - `coverage_stats`: compute per-section check coverage stats for appendix
   - NOT shared with Word renderer — per research anti-pattern: "Do NOT try to share rendering logic between Word and HTML"

2. **`_render_html_template(context) -> str`**
   - Set up Jinja2 Environment pointing to `templates/html/`
   - Register filters: format_currency, format_pct, na_if_none, risk_class, dim_display_name
   - Load `base.html.j2` which includes all section templates
   - Render and return HTML string

3. **`render_html_pdf(state, output_path, ds, chart_dir?) -> Path | None`**
   - Main entry point matching render_pdf signature
   - Build context, render HTML, write to temp file
   - Use Playwright to generate PDF:
     ```python
     with sync_playwright() as p:
         browser = p.chromium.launch(headless=True)
         page = browser.new_page()
         page.goto(f'file://{html_path}', wait_until='networkidle')  # wait for Tailwind CDN
         page.pdf(
             path=str(output_path),
             format='Letter',
             margin={'top': '0.75in', 'bottom': '0.75in',
                     'left': '0.65in', 'right': '0.65in'},
             print_background=True,
             display_header_footer=True,
             header_template='...',  # per research
             footer_template='... Page <span class="pageNumber"></span> of <span class="totalPages"></span> ...',
         )
         browser.close()
     ```
   - Cleanup temp HTML file
   - Fallback chain: Playwright -> existing WeasyPrint render_pdf -> None
   - Lazy import `from playwright.sync_api import sync_playwright` — if ImportError, try WeasyPrint fallback

4. **`_group_checks_by_section(check_results) -> dict[str, list[dict]]`**
   - Group check results by their section prefix (GOV., LIT., FIN., MKT., etc.)
   - Include content_type in each result for template dispatch

5. **`_compute_coverage_stats(check_results) -> dict[str, dict]`**
   - Per section: total checks, evaluated (TRIGGERED + CLEAR), skipped, info-only
   - Return dict for coverage appendix template

**Update `RenderStage` in `__init__.py`:**
- Import `render_html_pdf` from `html_renderer`
- Replace the PDF render call to use `render_html_pdf` as primary, with existing `render_pdf` (WeasyPrint) as fallback
- Pattern:
  ```python
  pdf_path = output_dir / f"{state.ticker}_worksheet.pdf"
  _render_secondary("PDF", render_html_pdf, state, pdf_path, ds)
  ```
