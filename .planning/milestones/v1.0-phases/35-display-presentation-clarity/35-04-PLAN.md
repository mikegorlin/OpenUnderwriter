**`base.html.j2`** (~150 lines):
- DOCTYPE html, charset utf-8, title with company name
- Tailwind CSS v4 CDN via `<script src="https://cdn.tailwindcss.com"></script>` — per research, this works in Playwright Chromium because it executes JS
- Custom `<style>` block linking to styles.css content (inline for PDF portability)
- Custom Tailwind config for Bloomberg colors: `tailwind.config = { theme: { extend: { colors: { navy: '#0B1D3A', gold: '#D4A843', ... } } } }`
- Header block: Dark navy bar with company name, ticker, "D&O Liability Underwriting Worksheet", generation date
- `{% block content %}{% endblock %}` for section injection
- Footer: "Confidential — For Internal Underwriting Use Only", page numbering placeholder
- Print media queries: `@media print { .page-break { page-break-before: always; } .no-print { display: none; } }`
- Import all component macros at top: `{% from "components/badges.html.j2" import traffic_light, density_indicator, confidence_marker %}`
- Font imports: Inter (headings), Georgia (body), system monospace (citations)
- `font-variant-numeric: tabular-nums` on data tables for aligned columns

**`components/badges.html.j2`** (~80 lines):
- `{% macro traffic_light(status, label="") %}` — TRIGGERED=red-700, CLEAR=emerald-600, SKIPPED=gray-400, INFO=blue-600, ELEVATED=amber-500
- `{% macro density_indicator(level) %}` — CRITICAL=red border-l-4 with "Critical Risk", ELEVATED=amber border-l-4 with "Elevated Concern", CLEAN=nothing
- `{% macro confidence_marker(confidence) %}` — Only renders for LOW confidence: small grey italic marker. MEDIUM/HIGH = no visible marker (per user decision)
- `{% macro tier_badge(tier) %}` — WIN=emerald, WANT=blue, WRITE=amber, WATCH=orange, WALK=red, NO_TOUCH=red-900

**`components/tables.html.j2`** (~120 lines):
- `{% macro data_table(headers, rows, id="") %}` — Full data table with alternating row shading (white/slate-50), navy header, borders
- `{% macro kv_table(pairs, title="") %}` — Key-value table with 35% label width, lighter header
- `{% macro multi_column_grid(cols=2, items=[]) %}` — CSS Grid wrapper: `grid grid-cols-{{ cols }} gap-4`
- `{% macro conditional_cell(value, direction="neutral", threshold_pct=10) %}` — Applies risk coloring: >threshold% deterioration=red, >threshold% improvement=blue, else neutral. Uses CSS classes not inline styles
- `{% macro financial_row(label, value, yoy_change, direction) %}` — Single financial row with conditional YoY formatting

**`components/callouts.html.j2`** (~60 lines):
- `{% macro discovery_box(title, content) %}` — Distinct "Discovery" callout with gold left border, light gold background — per user decision on blind spot callout boxes
- `{% macro warning_box(description, do_context="") %}` — Amber warning with optional D&O context note — for ELEVATED density items
- `{% macro do_context(text) %}` — Small italic navy text explaining "why this matters for D&O" — per user decision on ELEVATED indicators
- `{% macro gap_notice(field_name, reason) %}` — Grey "Data not available" notice with explanation — per user decision on per-subsection gap notices

**`components/charts.html.j2`** (~30 lines):
- `{% macro embed_chart(chart_name, charts_dict, alt_text="", full_width=true) %}` — Embeds base64 PNG chart image from charts_dict. Full-width or half-width via class. Fallback text if chart not available.

**`components/narratives.html.j2`** (~40 lines):
- `{% macro section_narrative(text, ai_generated=true) %}` — Renders narrative paragraph with optional "AI Assessment" label per DATA-14. Georgia font, professional spacing.
- `{% macro evidence_chain(items) %}` — Bulleted evidence list for INFERENCE_PATTERN checks — per user decision on "written narrative + bulleted evidence list"

Keep each component file under 200 lines per research guidance.

Custom CSS that supplements Tailwind for Bloomberg/S&P quality:

Colors (matching research recommendations):
- Primary Navy: #0B1D3A
- Accent Gold: #D4A843
- Risk Red: #B91C1C (Tailwind red-700)
- Caution Amber: #D97706 (Tailwind amber-600)
- Positive Blue: #1D4ED8 (Tailwind blue-700, NOT green per research)
- Neutral Gray: #6B7280
- Background: #FFFFFF body, #F8FAFC alternating rows
- Text: #111827 body, #6B7280 secondary

Typography:
- Headings: Inter, system-ui, sans-serif — `font-weight: 600`
- Body: Georgia, 'Times New Roman', serif — `line-height: 1.6`
- Data/Tables: `font-variant-numeric: tabular-nums` — aligned columns
- Citations: 'JetBrains Mono', monospace, 7pt

Financial table styling:
- `.risk-critical { color: #B91C1C; font-weight: 600; }` — red
- `.risk-elevated { color: #D97706; }` — amber
- `.risk-positive { color: #1D4ED8; }` — blue (not green)
- `.risk-neutral { color: #6B7280; }` — grey

Print styles:
- Page margins: 0.75in top/bottom, 0.65in left/right
- `@page { size: letter; }`
- `.page-break { page-break-before: always; }`
- Avoid orphans/widows in narrative text
- Hide interactive elements
