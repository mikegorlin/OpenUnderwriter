Concrete implementation:
```python
# After the existing rev_val / ni_val extraction:
# Find line items for YoY
rev_item_yoy = next(
    (it for it in stmts.income_statement.line_items if "revenue" in it.label.lower()),
    None,
)
ni_item_yoy = next(
    (it for it in stmts.income_statement.line_items if "net_income" in it.label.lower().replace(" ", "_")),
    None,
)
result["revenue_yoy"] = (
    format_change_indicator(rev_val, prior_rev)
    if rev_val is not None and prior_rev is not None and prior_rev != 0
    else ""
)
result["net_income_yoy"] = (
    format_change_indicator(ni_val, prior_ni)
    if ni_val is not None and prior_ni is not None and prior_ni != 0
    else ""
)
```

Note: Use `format_change_indicator(current, prior)` from `formatters.py` which already produces "+12.3%" / "-5.7%" format. Import it at the top alongside the existing formatter imports.

result["revenue_yoy"] = f"{rev_item_yoy.yoy_change:+.1f}%" if rev_item_yoy and rev_item_yoy.yoy_change is not None else ""
