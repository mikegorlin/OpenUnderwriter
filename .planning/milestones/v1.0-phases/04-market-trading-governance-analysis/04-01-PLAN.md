Add three new public functions:

1. `fetch_proxy_text(filings_metadata: dict[str, Any], cik: str) -> dict[str, str]`
   - Find latest DEF 14A filing via _get_latest_filing()
   - Fetch HTML via sec_get_text()
   - Strip HTML tags using existing _strip_html()
   - Extract key sections via regex: "EXECUTIVE COMPENSATION", "DIRECTOR COMPENSATION", "SECURITY OWNERSHIP", "CORPORATE GOVERNANCE", "PROPOSAL", "RELATED PARTY", "CERTAIN RELATIONSHIPS"
   - Use same section extraction approach as _extract_10k_sections() but with proxy-specific markers
   - Return dict with keys: "proxy_compensation", "proxy_governance", "proxy_ownership", "proxy_proposals", "proxy_related_party", "proxy_full"
   - Truncate each section to MAX_SECTION_CHARS

2. `fetch_form4_xml(filings_metadata: dict[str, Any], cik: str, max_filings: int = 20) -> list[dict[str, Any]]`
   - Get up to max_filings Form 4 filing metadata entries from filings_metadata["4"]
   - For each: fetch the primary document URL via sec_get_text()
   - Return list of dicts: {"accession": str, "filing_date": str, "xml_text": str}
   - Rate-limit per SEC rules (sec_get_text handles this)

3. `fetch_8k_text(filings_metadata: dict[str, Any], cik: str, item_filter: str = "5.02") -> list[dict[str, Any]]`
   - Get 8-K filings from metadata
   - Fetch each 8-K primary document HTML
   - Extract text near the specified item number (default "5.02" for leadership changes)
   - Return list: {"accession": str, "filing_date": str, "item": str, "text": str}
   - Limit to 10 most recent 8-Ks to bound HTTP calls
