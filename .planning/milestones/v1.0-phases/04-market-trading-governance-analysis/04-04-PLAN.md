    state: AnalysisState,

**Internal helpers:**

1. `_find_single_day_drops(history: dict[str, Any], threshold_pct: float = -5.0) -> list[StockDropEvent]`
   - Iterate Close prices from history_1y (or history_5y truncated to 18 months)
   - Compute daily return: (close[i] - close[i-1]) / close[i-1] * 100
   - Flag days with return <= threshold (-5%, -10%, -20% for severity tiering)
   - For each drop, create StockDropEvent with date, drop_pct, close_price
   - Handle None/NaN values defensively

2. `_find_multi_day_drops(history: dict[str, Any], periods: list[int], thresholds: dict[int, float]) -> list[StockDropEvent]`
   - periods = [2, 5, 20] (trading days)
   - thresholds = {2: -10.0, 5: -15.0, 20: -25.0}
   - For each period N: compute rolling N-day returns, flag those below threshold
   - Create StockDropEvent with drop_type="MULTI_DAY", period_days=N
   - Deduplicate overlapping windows (keep worst)

3. `_compute_sector_comparison(drop: StockDropEvent, sector_history: dict[str, Any]) -> StockDropEvent`
   - If state has sector ETF data (from peer group construction), compute sector return on same date(s)
   - Set sector_return_pct and is_company_specific flag
   - If no sector data available, leave sector fields as None

4. `_compute_performance_metrics(history_1y: dict, history_5y: dict, info: dict) -> StockPerformance`
   - current_price from info dict
   - high_52w, low_52w from info dict
   - decline_from_high_pct = (current - high) / high * 100
   - returns_1y, returns_3y, returns_5y, returns_ytd from history data (first-to-last return)
   - beta from info dict
   - volatility_90d: std(daily_returns[-90:]) * sqrt(252) for annualization
   - max_drawdown_1y: max peak-to-trough decline in history_1y
   - sector_relative_performance from info or computed

5. `_attribute_triggers(drops: list[StockDropEvent], filings: dict, market_data: dict) -> list[StockDropEvent]`
   - For each drop, search 8-K filings within +/-2 business days for possible trigger
   - Search earnings_dates for earnings-day drops
   - Search news for matching dates
   - Set trigger_event field; leave as None if no attribution found
