  <name>Task 1: Create market_events.py with typed sub-models</name>
  <files>src/do_uw/models/market_events.py</files>
  <action>
Create new file with typed Pydantic models for SECT4 data:

1. **StockDropEvent** (SECT4-03):
   - date: SourcedValue[str] | None -- date of drop
   - drop_pct: SourcedValue[float] | None -- percentage decline
   - drop_type: str = "" -- "SINGLE_DAY" or "MULTI_DAY" (with period_days for multi-day)
   - period_days: int = 1 -- 1 for single-day, 2/5/20 for multi-day
   - sector_return_pct: SourcedValue[float] | None -- sector ETF return on same day(s)
   - is_company_specific: bool = False -- True if sector ETF was flat/positive
   - trigger_event: SourcedValue[str] | None -- attributed cause (8-K, earnings, news)
   - ddl_window_start: str = "" -- estimated DDL window start date
   - close_price: float | None = None -- closing price on drop date

2. **StockDropAnalysis** (SECT4-03 container):
   - single_day_drops: list[StockDropEvent] -- drops >5% in one day
   - multi_day_drops: list[StockDropEvent] -- declines >10% over 2-20 days
   - analysis_period_months: int = 18
   - worst_single_day: StockDropEvent | None
   - worst_multi_day: StockDropEvent | None

3. **InsiderTransaction** (SECT4-04):
   - insider_name: SourcedValue[str] | None
   - insider_title: SourcedValue[str] | None
   - transaction_date: SourcedValue[str] | None
   - transaction_type: str = "" -- "SALE", "PURCHASE", "GRANT", "EXERCISE", etc.
   - transaction_code: str = "" -- raw Form 4 code: S, P, A, M, F, G, D
   - shares: SourcedValue[int] | None
   - price_per_share: SourcedValue[float] | None
   - total_value: SourcedValue[float] | None
   - shares_remaining: SourcedValue[int] | None
   - is_10b5_1: SourcedValue[bool] | None -- from AFF10B5ONE element
   - is_discretionary: bool = False -- True if sale without 10b5-1 plan

4. **InsiderClusterEvent** (SECT4-04):
   - start_date: str = ""
   - end_date: str = ""
   - insider_count: int = 0
   - insiders: list[str] = Field(default_factory=lambda: [])
   - total_value: float = 0.0
   - window_days: int = 30

5. **InsiderTradingAnalysis** (SECT4-04 container):
   - transactions: list[InsiderTransaction] = Field(default_factory=lambda: [])
   - cluster_events: list[InsiderClusterEvent] = Field(default_factory=lambda: [])
   - net_buying_selling: SourcedValue[str] | None -- NET_BUYING, NET_SELLING, NEUTRAL
   - total_sold_value: SourcedValue[float] | None
   - total_bought_value: SourcedValue[float] | None
   - pct_10b5_1: SourcedValue[float] | None -- percentage of sales under 10b5-1
   - aggregate_ownership: SourcedValue[float] | None
   - pledged_shares_value: SourcedValue[float] | None

6. **EarningsQuarterRecord** (SECT4-06 single quarter):
   - quarter: str = "" -- "Q1 2024"
   - guidance_low: float | None = None
   - guidance_high: float | None = None
   - guidance_midpoint: float | None = None
   - revision: str = "" -- "RAISED", "LOWERED", "NARROWED", "NONE"
   - actual_eps: float | None = None
   - result: str = "" -- "BEAT", "MISS", "MEET"
   - miss_magnitude_pct: float | None = None -- miss as % of guided midpoint
   - stock_reaction_pct: float | None = None -- stock move on earnings day

7. **EarningsGuidanceAnalysis** (SECT4-06 container):
   - quarters: list[EarningsQuarterRecord] = Field(default_factory=lambda: [])
   - beat_rate: SourcedValue[float] | None
   - consecutive_miss_count: int = 0
   - guidance_withdrawals: int = 0
   - philosophy: SourcedValue[str] | None -- "CONSERVATIVE", "AGGRESSIVE", "NO_GUIDANCE"

8. **AnalystSentimentProfile** (SECT4-07):
   - coverage_count: SourcedValue[int] | None
   - consensus: SourcedValue[str] | None -- "BUY", "HOLD", "SELL"
   - recommendation_mean: SourcedValue[float] | None -- 1.0-5.0
   - target_mean: SourcedValue[float] | None
   - target_median: SourcedValue[float] | None
   - target_high: SourcedValue[float] | None
   - target_low: SourcedValue[float] | None
   - recent_upgrades_90d: SourcedValue[int] | None
   - recent_downgrades_90d: SourcedValue[int] | None
   - estimate_revisions_up: SourcedValue[int] | None
   - estimate_revisions_down: SourcedValue[int] | None

9. **CapitalMarketsOffering** (SECT4-08 single offering):
   - offering_type: str = "" -- "EQUITY", "DEBT", "CONVERTIBLE", "ATM", "PIPE"
   - filing_type: str = "" -- "S-3", "S-1", "424B"
   - date: str = ""
   - amount: SourcedValue[float] | None
   - section_11_window_end: str = "" -- date + 3 years

10. **CapitalMarketsActivity** (SECT4-08 container):
    - shelf_registrations: list[CapitalMarketsOffering] = Field(default_factory=lambda: [])
    - offerings_3yr: list[CapitalMarketsOffering] = Field(default_factory=lambda: [])
    - has_atm_program: SourcedValue[bool] | None
    - convertible_securities: list[SourcedValue[dict[str, Any]]] = Field(default_factory=lambda: [])
    - active_section_11_windows: int = 0

11. **AdverseEventScore** (SECT4-09):
    - total_score: SourcedValue[float] | None
    - event_count: int = 0
    - severity_breakdown: dict[str, float] = Field(default_factory=dict)
    - peer_rank: SourcedValue[int] | None
    - peer_percentile: SourcedValue[float] | None

All models use: model_config = ConfigDict(frozen=False), SourcedValue from do_uw.models.common, Field with default_factory=lambda: [] for lists.
