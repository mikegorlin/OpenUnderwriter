**board_governance.py:**

1. `extract_board_governance(state: AnalysisState) -> tuple[tuple[list[BoardForensicProfile], GovernanceQualityScore], ExtractionReport]`
   - Orchestrates board profile extraction and governance scoring

2. `_extract_board_from_proxy(proxy_text: dict[str, str], state: AnalysisState) -> list[BoardForensicProfile]`
   - Parse proxy_governance section for director names, tenure, independence status
   - Pattern: director names followed by "independent" or "non-independent" designation
   - Extract committee memberships from committee section
   - Count other_boards from bio text ("serves on the board of X, Y")
   - is_overboarded = len(other_boards) + 1 >= threshold (default 4)
   - **Prior litigation search:** For each director, search state.acquired_data.litigation_data and state.acquired_data.web_search_results/blind_spot_results for name matches (same approach as leadership_profiles.py _search_prior_litigation). Populate prior_litigation field with matches. LOW confidence.
   - Fallback to existing state.extracted.governance.board for basic metrics

3. `_compute_governance_score(board_profiles: list[BoardForensicProfile], compensation: CompensationAnalysis, weights: dict, thresholds: dict) -> GovernanceQualityScore`
   - Load weights from governance_weights.json
   - Score each dimension on 0-10 scale per research thresholds:
     - independence_score: >75% = 10, 50-75% = 7, <50% = 3
     - ceo_chair_score: separated = 10, duality = 3
     - refreshment_score: 2+ new in 3yr = 10, 1 = 7, 0 = 3
     - overboarding_score: 0 overboarded = 10, 1 = 7, 2+ = 3
     - committee_score: all key committees (audit, comp, nom/gov) = 10, missing any = 5
     - say_on_pay_score: >90% = 10, 70-90% = 7, <70% = 3
     - tenure_score: avg 5-10yr = 10, <5 or >15 = 5
   - total_score = weighted sum normalized to 0-100

ExtractionReport expected: ["board_profiles", "independence_ratio", "overboarding", "committee_structure", "governance_score"]

**ownership_structure.py:**

1. `extract_ownership(state: AnalysisState) -> tuple[OwnershipAnalysis, ExtractionReport]`
   - Extract from multiple sources:
     - yfinance institutional_holders -> top holders, institutional_pct
     - yfinance info dict -> insider ownership percentage
     - DEF 14A proxy_ownership section -> ownership table, dual-class
     - SEC 13D/13G filings metadata -> activist filings

2. `_extract_from_institutional_holders(holders_data: dict) -> tuple[list, float]`
   - Parse institutional_holders dict from yfinance
   - Extract top 10 holders with name, shares, value, date
   - Compute institutional_pct from info dict or sum of holder percentages

3. `_check_for_activists(holders: list, activists: list[str]) -> list[str]`
   - Load known_activists from activist_investors.json
   - Match holder names against activist list (case-insensitive, partial match)
   - Return list of matched activist names

4. `_extract_13d_filings(filings: dict) -> tuple[list, list]`
   - Parse SC 13D and SC 13G filing metadata from acquired filings
   - Identify 13D filings (activist/control intent) in trailing 24 months
   - Identify 13G-to-13D conversions (passive to active -- warning signal)
   - A conversion is when the same filer has both a 13G and a later 13D

5. `_assess_activist_risk(ownership: OwnershipAnalysis) -> str`
   - "HIGH" if: any 13D filing, any known activist in holders, or 13G-to-13D conversion
   - "MEDIUM" if: institutional_pct > 85% (highly concentrated), or proxy contests in 3yr
   - "LOW" otherwise

6. `_extract_dual_class(proxy_text: dict, info: dict) -> tuple[bool, float | None, float | None]`
   - Check proxy text for "dual-class", "Class A", "Class B" voting structure
   - If dual-class: extract voting control % and economic ownership %
   - Fallback to info dict for basic structure detection

ExtractionReport expected: ["institutional_pct", "insider_pct", "top_holders", "activist_check", "13d_filings", "dual_class", "activist_risk"]
