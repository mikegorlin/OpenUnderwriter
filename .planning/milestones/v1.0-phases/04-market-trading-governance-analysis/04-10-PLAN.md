**extract_market.py:**

```python
def run_market_extractors(
    state: AnalysisState,
    reports: list[ExtractionReport],
) -> MarketSignals:
```

Calls SECT4 extractors in dependency order:
1. extract_stock_performance(state) -> StockPerformance, StockDropAnalysis, report
2. extract_insider_trading(state) -> InsiderTradingAnalysis, report
3. extract_short_interest(state) -> ShortInterestProfile, report
4. extract_earnings_guidance(state) -> EarningsGuidanceAnalysis, report
5. extract_analyst_sentiment(state) -> AnalystSentimentProfile, report
6. extract_capital_markets(state) -> CapitalMarketsActivity, report
7. compute_adverse_event_score(state) -> AdverseEventScore, report (MUST be last -- depends on all above)

Each extractor call wrapped in try/except. On failure: log warning, create failed ExtractionReport, continue with remaining extractors.

Assembles MarketSignals from all outputs:
- market.stock = stock_performance
- market.stock_drops = stock_drop_analysis
- market.insider_analysis = insider_trading_analysis
- market.short_interest = short_interest_profile
- market.earnings_guidance = earnings_guidance_analysis
- market.analyst = analyst_sentiment
- market.capital_markets = capital_markets_activity
- market.adverse_events = adverse_event_score

Appends all ExtractionReports to the reports list.
Returns the assembled MarketSignals.

**extract_governance.py:**

```python
def run_governance_extractors(
    state: AnalysisState,
    reports: list[ExtractionReport],
) -> GovernanceData:
```

Calls SECT5 extractors in dependency order:
1. extract_leadership_profiles(state) -> LeadershipStability, report
2. extract_compensation(state) -> CompensationAnalysis, report
3. extract_board_governance(state) -> (list[BoardForensicProfile], GovernanceQualityScore), report
4. extract_ownership(state) -> OwnershipAnalysis, report
5. extract_sentiment(state) -> SentimentProfile, report
6. assess_narrative_coherence(state) -> NarrativeCoherence, report (MUST be last -- references other extracted data)

Each call wrapped in try/except. Same pattern as market sub-orchestrator.

Assembles GovernanceData from all outputs:
- governance.leadership = leadership_stability
- governance.comp_analysis = compensation_analysis
- governance.board_forensics = board_profiles
- governance.governance_score = governance_quality_score
- governance.ownership = ownership_analysis
- governance.sentiment = sentiment_profile
- governance.narrative_coherence = narrative_coherence

**Governance summary narrative (SECT5-01):**
After all extractors complete, generate governance_summary as a rule-based text synthesis (same pattern as financial health narrative from 03-07). The narrative should synthesize 5 dimensions:

1. **Leadership stability assessment:** "[Company] has [stable/concerning] leadership with [N] executives, average tenure of [X] years. [If departures: 'Notable: [N] departures in trailing 18 months including [titles].' If CFO departure: 'Red flag: CFO departure [date] classified as [planned/unplanned].']"

2. **Board quality highlights:** "Board governance scores [high/moderate/low] ([score]/100). Independence ratio: [X]%. [If overboarded: '[N] directors serve on 4+ boards.'] [If duality: 'CEO serves as board chair.'] [If low say-on-pay: 'Say-on-pay support at [X]%, below typical threshold.']"

3. **Compensation concerns:** "[If high pay ratio: 'CEO pay ratio of [X]:1 exceeds sector median.'] [If low say-on-pay: 'Shareholder support for compensation at [X]%.'] [If no clawback: 'No clawback policy beyond Dodd-Frank minimum.'] [If related party: '[N] related party transactions disclosed.']"

4. **Ownership/activist risks:** "Institutional ownership: [X]%. [If activist: 'Known activist [name] holds position.'] [If 13D: '[N] Schedule 13D filings in trailing 24 months.'] [If dual-class: 'Dual-class structure with [X]% voting control vs [Y]% economic ownership.'] Activist risk assessment: [LOW/MEDIUM/HIGH]."

5. **Sentiment/coherence flags:** "Management tone trajectory: [IMPROVING/STABLE/DETERIORATING]. [If misalignment: 'Narrative coherence concern: [specific flag, e.g., insider selling diverges from management optimism].'] [If Glassdoor: 'Employee sentiment: [rating]/5.0.']"

Store as SourcedValue[str] with LOW confidence and source "Rule-based synthesis of governance extraction results". If insufficient data for any dimension, omit that section rather than generating misleading text.

Both sub-orchestrators should be under 300 lines each (focused delegation, no business logic beyond narrative generation).
