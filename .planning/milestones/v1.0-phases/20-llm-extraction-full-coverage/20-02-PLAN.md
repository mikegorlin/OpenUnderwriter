<task type="auto">
  <name>Task 1: Create ten_k_converters.py with all conversion functions</name>
  <files>src/do_uw/stages/extract/ten_k_converters.py</files>
  <action>
    Create `src/do_uw/stages/extract/ten_k_converters.py` following the established Phase 19 converter pattern (see llm_governance.py for reference).

    Use `_LLM_SOURCE = "10-K (LLM)"` and Confidence.HIGH for all fields.
    Import from `sourced.py`: `sourced_str`, `sourced_float`, `sourced_int`, `now`.
    Import `TenKExtraction` from the schemas.
    Import `SourcedValue`, `Confidence` from `do_uw.models.common`.

    **Item 1 (Business) converters:**

    1. `convert_business_description(extraction: TenKExtraction) -> SourcedValue[str] | None`
       - Returns `sourced_str(extraction.business_description, _LLM_SOURCE, Confidence.HIGH)` if non-empty, else None

    2. `convert_revenue_segments(extraction: TenKExtraction) -> list[SourcedValue[dict[str, str]]]`
       - Parses each segment string (format "Cloud Services: 60%") into `{"segment": name, "percentage": pct}`
       - Returns empty list if no segments

    3. `convert_geographic_footprint(extraction: TenKExtraction) -> list[SourcedValue[dict[str, str]]]`
       - Parses each region string (format "United States: 55%") into `{"region": name, "percentage": pct}`
       - Returns empty list if no regions

    4. `convert_customer_concentration(extraction: TenKExtraction) -> list[SourcedValue[str]]`
       - Wraps each concentration string in SourcedValue

    5. `convert_supplier_concentration(extraction: TenKExtraction) -> list[SourcedValue[str]]`
       - Same pattern as customer_concentration

    6. `convert_operational_complexity_flags(extraction: TenKExtraction) -> dict[str, SourcedValue[bool]]`
       - Returns dict with "has_dual_class" and "has_vie" if extracted, empty dict if both None
       - Uses a local helper `_sourced_bool(value: bool) -> SourcedValue[bool]` (same pattern as llm_governance.py)

    7. `convert_employee_count(extraction: TenKExtraction) -> SourcedValue[int] | None`
       - Returns `sourced_int(extraction.employee_count, _LLM_SOURCE, Confidence.HIGH)` if non-None

    8. `convert_competitive_position(extraction: TenKExtraction) -> SourcedValue[str] | None`
       - Returns sourced_str if non-None

    9. `convert_regulatory_environment(extraction: TenKExtraction) -> SourcedValue[str] | None`
       - Returns sourced_str if non-None

    **Item 7 (MD&A) converters:**

    10. `convert_mda_qualitative(extraction: TenKExtraction) -> dict[str, SourcedValue[str] | list[SourcedValue[str]] | None]`
        - Returns a dict with keys: "revenue_trend", "margin_trend", "key_financial_concerns", "guidance_language"
        - Each is sourced_str if non-None
        - "critical_accounting_estimates" and "non_gaap_measures" are lists of sourced_str

    **Item 8 (Footnotes) converters:**

    11. `convert_debt_enrichment(extraction: TenKExtraction) -> dict[str, SourcedValue[str] | list[SourcedValue[str]] | None]`
        - Returns dict with "debt_instruments" (list of sourced_str), "credit_facility_detail" (sourced_str), "covenant_status" (sourced_str), "tax_rate_notes" (sourced_str)
        - These SUPPLEMENT XBRL -- they are qualitative context, never overriding XBRL numbers

    12. `convert_stock_comp_detail(extraction: TenKExtraction) -> SourcedValue[str] | None`
        - Returns sourced_str for stock_comp_detail if non-None

    **Item 9A (Controls) converters:**

    13. `convert_controls_assessment(extraction: TenKExtraction) -> dict[str, Any]`
        - Returns dict with:
          - "has_material_weakness": SourcedValue[bool] | None
          - "material_weakness_detail": list[SourcedValue[str]]
          - "significant_deficiencies": list[SourcedValue[str]]
          - "remediation_status": SourcedValue[str] | None
          - "auditor_attestation": SourcedValue[str] | None
          - "auditor_name": SourcedValue[str] | None
          - "auditor_tenure_years": SourcedValue[int] | None
        - Use _sourced_bool for has_material_weakness
        - Each list entry is a sourced_str

    **LINE COUNT TARGET:** Start with all converters in one file. If it exceeds 450 lines, split into `ten_k_converters.py` (Items 1/7, ~200 lines) and `ten_k_converters_financial.py` (Items 8/9A, ~200 lines). The split boundary is natural: Items 1/7 are qualitative text while Items 8/9A are financial/controls.

    **Helper functions** (private):
    - `_sourced_bool(value: bool) -> SourcedValue[bool]` -- same pattern as llm_governance.py
    - `_parse_colon_pair(text: str) -> dict[str, str]` -- parses "Name: Value" into {"name": left_stripped, "value": right_stripped}
