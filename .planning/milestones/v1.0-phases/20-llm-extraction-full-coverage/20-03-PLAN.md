  <action>
    Create `src/do_uw/stages/extract/eight_k_converter.py` (~200-300 lines).

    Use `_LLM_SOURCE = "8-K (LLM)"` and Confidence.HIGH.

    **Key design:** 8-K is multi-instance. Each function takes `list[EightKExtraction]` and aggregates across all of them. Each 8-K typically covers one event type.

    **Converter functions:**

    1. `convert_departures(extractions: list[EightKExtraction]) -> list[dict[str, SourcedValue[str] | SourcedValue[bool] | None]]`
       - Iterates all extractions; for each with `departing_officer` non-None, creates a departure dict:
         - "name": sourced_str(departing_officer)
         - "title": sourced_str(departing_officer_title) if non-None
         - "reason": sourced_str(departure_reason) if non-None
         - "successor": sourced_str(successor) if non-None
         - "is_termination": _sourced_bool(is_termination) if non-None
         - "event_date": sourced_str(event_date) if non-None

    2. `convert_agreements(extractions: list[EightKExtraction]) -> list[dict[str, SourcedValue[str] | SourcedValue[float] | None]]`
       - For each with `agreement_type` non-None:
         - "type": sourced_str(agreement_type)
         - "counterparty": sourced_str(counterparty) if non-None
         - "summary": sourced_str(agreement_summary) if non-None
         - "event_date": sourced_str(event_date) if non-None

    3. `convert_acquisitions(extractions: list[EightKExtraction]) -> list[dict[str, SourcedValue[str] | SourcedValue[float] | None]]`
       - For each with `transaction_type` non-None:
         - "type": sourced_str(transaction_type)
         - "target": sourced_str(target_name) if non-None
         - "value": sourced_float(transaction_value, _LLM_SOURCE, Confidence.HIGH) if non-None
         - "event_date": sourced_str(event_date) if non-None

    4. `convert_restatements(extractions: list[EightKExtraction]) -> list[dict[str, SourcedValue[str] | list[SourcedValue[str]] | None]]`
       - For each with non-empty `restatement_periods`:
         - "periods": [sourced_str(p) for p in restatement_periods]
         - "reason": sourced_str(restatement_reason) if non-None
         - "event_date": sourced_str(event_date) if non-None

    5. `convert_earnings_events(extractions: list[EightKExtraction]) -> list[dict[str, SourcedValue[str] | SourcedValue[float] | None]]`
       - For each with `revenue` or `eps` non-None:
         - "revenue": sourced_float(revenue) if non-None
         - "eps": sourced_float(eps) if non-None
         - "guidance_update": sourced_str(guidance_update) if non-None
         - "event_date": sourced_str(event_date) if non-None

    **Private helpers:**
    - `_sourced_bool(value: bool) -> SourcedValue[bool]`

    **tests/test_eight_k_converter.py** -- Unit tests:

    Create a `_make_departure_8k()`, `_make_agreement_8k()`, `_make_restatement_8k()` helper for each event type.

    Tests:

    - `test_convert_departures_empty` -- Empty list -> empty list
    - `test_convert_departures_skip_no_officer` -- 8-K without departing_officer is skipped
    - `test_convert_departures_with_title` -- departing_officer_title populated correctly
    - `test_convert_agreements` -- Agreement fields mapped correctly
    - `test_convert_acquisitions_with_value` -- Transaction value as sourced_float
    - `test_convert_restatements` -- Restatement periods as list of sourced_str
    - `test_convert_restatements_empty_periods` -- Empty restatement_periods -> skipped
    - `test_convert_earnings_events` -- Revenue and EPS as sourced_float
    - `test_mixed_event_types` -- List with departure + agreement -> each converter picks its events
  </action>

  <action>
    Create `src/do_uw/stages/extract/proxy_ownership_converter.py` (~100-200 lines).

    Use `_LLM_SOURCE = "DEF 14A (LLM)"` and Confidence.HIGH.

    **Converter functions:**

    1. `convert_top_holders(extraction: DEF14AExtraction) -> list[SourcedValue[dict[str, str]]]`
       - Parses each string in `extraction.top_5_holders` (format: "Vanguard Group: 8.2%")
       - Returns list of SourcedValue[dict] with {"name": "Vanguard Group", "percentage": "8.2%"}
       - Handle edge cases: no colon separator (use full string as name), empty list
       - Returns empty list if top_5_holders is empty

    2. `convert_insider_ownership(extraction: DEF14AExtraction) -> SourcedValue[float] | None`
       - Returns sourced_float(officers_directors_ownership_pct) if non-None
       - Returns None otherwise

    3. `convert_proxy_ownership_summary(extraction: DEF14AExtraction) -> dict[str, SourcedValue[float] | list[SourcedValue[dict[str, str]]] | None]`
       - Convenience function combining both:
         - "insider_pct": convert_insider_ownership result
         - "top_holders": convert_top_holders result

    **tests/test_proxy_ownership_converter.py** -- Unit tests:

    Create `_make_proxy()` helper returning a DEF14AExtraction with ownership data:
    ```python
    DEF14AExtraction(
        officers_directors_ownership_pct=21.3,
        top_5_holders=["Elon Musk: 21.3%", "Vanguard Group: 6.1%", "BlackRock: 4.8%"],
    )
    ```

    Tests:
    - `test_convert_top_holders` -- 3 holders -> 3 SourcedValue dicts with name and percentage
    - `test_convert_top_holders_empty` -- Empty list -> empty list
    - `test_convert_top_holders_no_colon` -- "Elon Musk 21%" -> name is full string
    - `test_convert_insider_ownership` -- 21.3 -> SourcedValue(21.3) with DEF 14A (LLM) source
    - `test_convert_insider_ownership_none` -- None -> None
    - `test_convert_proxy_ownership_summary` -- Returns both fields populated
    - `test_convert_proxy_ownership_summary_partial` -- Only insider_pct, no top_5_holders
  </action>
