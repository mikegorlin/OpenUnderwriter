**Step 2: Create 5 new playbooks following the exact same structure as existing ones**

Use the _check() helper function pattern from playbook_data.py. Each playbook dict must have:
- id: Unique string (e.g., "CPG_CONSUMER", "INDUSTRIALS_MFG", "MEDIA_ENTERTAINMENT", "REITS_REAL_ESTATE", "TRANSPORTATION_RAIL")
- name: Display name
- description: 1-2 sentence description
- sic_ranges: list of {"low": int, "high": int} dicts. Use standard SIC ranges:
  - CPG: 2000-2099 (food), 2100-2199 (tobacco), 2800-2899 (chemicals/soap)
  - Industrials/Mfg: 3400-3499 (fabricated metals), 3500-3599 (industrial machinery), 3600-3699 (electronic equipment), 3700-3799 (transportation equipment)
  - Media/Entertainment: 2700-2799 (publishing), 4800-4899 (communications), 7810-7819 (motion pictures), 7900-7999 (amusement)
  - REITs: 6500-6553 (real estate)
  - Transportation: 4000-4099 (railroads), 4100-4199 (transit), 4200-4299 (trucking), 4400-4499 (water transport), 4500-4599 (air transport)
- naics_prefixes: list of string NAICS prefixes
- industry_checks: list of check dicts using _check() -- extract 8-10 checks per industry from the supplement content
- claim_theories: list of dicts with id, name, description, factors_affected, historical_examples
- meeting_questions: list of dicts with question, category, priority
- scoring_adjustments: dict with factor adjustments (e.g., {"F1_litigation": 1.2} for enhanced weight)

**For each industry, extract checks from the supplement's key risk factors, red flags, and underwriting triggers.** Focus on checks that are INDUSTRY-SPECIFIC and NOT already covered by the generic 359 checks. Examples:
- CPG: M&A-driven goodwill impairment, zero-based budgeting damage, private label competition, brand equity erosion
- Industrials: Product safety cascading to securities liability, cyclical revenue manipulation, supply chain/tariff disclosure, asbestos/environmental legacy
- Media: Streaming subscriber metric manipulation, content amortization accounting, defamation exposure, cord-cutting revenue cliff
- REITs: AFFO methodology divergence, external management conflicts, dividend payout sustainability, sector concentration risk (office vs. data center)
- Transportation/Rail: PSR safety trade-off disclosure, deferred maintenance concealment, operating ratio manipulation, derailment/incident liability cascade

```python
def test_ingest_supplements():
    """Verify supplement documents can be ingested."""
    store = KnowledgeStore(db_path=None)
    from do_uw.knowledge.ingestion import ingest_document, DocumentType
    from pathlib import Path

    supplements = [
        "cpg_industry_module_supplement.md",
        "industrials_manufacturing_industry_module_supplement.md",
        "media_entertainment_industry_module_supplement.md",
        "reits_real_estate_industry_module_supplement_v2.md",
        "transportation_freight_rail_industry_module_supplement.md",
    ]
    old_uw = Path("Old Underwriter")
    total_checks = 0
    total_notes = 0
    for name in supplements:
        path = old_uw / name
        if path.exists():
            result = ingest_document(store, path, DocumentType.INDUSTRY_ANALYSIS)
            total_checks += result.checks_created
            total_notes += result.notes_added
            assert result.errors == [], f"Errors ingesting {name}: {result.errors}"

    # Should extract meaningful content from all 5 supplements
    assert total_checks + total_notes > 0, "No content extracted from supplements"
```

Note: This test uses real file paths so it will only pass when the Old Underwriter/ directory exists. Mark with `@pytest.mark.skipif` if the directory doesn't exist, using `Path("Old Underwriter").exists()` check.
