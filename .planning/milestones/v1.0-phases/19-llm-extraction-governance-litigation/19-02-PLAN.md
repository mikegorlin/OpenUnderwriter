<objective>
Build the governance converter module that maps DEF14AExtraction fields to GovernanceData sub-models.

Purpose: This is the core value-add of Phase 19 for Section 5 (Governance). The LLM reads actual proxy statement text and extracts structured data about directors, compensation, board composition, and ownership. This converter translates those flat LLM fields into the rich domain models (BoardForensicProfile, CompensationAnalysis, BoardProfile, CompensationFlags, OwnershipAnalysis) that the scoring engine and worksheet renderer consume. Every field gets HIGH confidence and 'DEF 14A (LLM)' source.

Output: `llm_governance.py` converter module + comprehensive unit tests.

  <name>Task 1: Create governance converter module</name>
  <files>src/do_uw/stages/extract/llm_governance.py</files>
  <action>
    Create `llm_governance.py` in `src/do_uw/stages/extract/`. Must stay under 500 lines.

    **Constants:**
    - `_LLM_SOURCE = "DEF 14A (LLM)"`

    **Functions to implement:**

    1. `convert_directors(extraction: DEF14AExtraction) -> list[BoardForensicProfile]`
       - Iterate `extraction.directors`, create `BoardForensicProfile` for each:
         - `name`: `sourced_str(d.name, _LLM_SOURCE, Confidence.HIGH)` if d.name
         - `tenure_years`: `sourced_float(d.tenure_years, ...)` if not None
         - `is_independent`: `SourcedValue[bool](value=d.independent, source=_LLM_SOURCE, confidence=Confidence.HIGH, as_of=now())` if not None
         - `committees`: `list(d.committees)` direct copy
         - `other_boards`: `[sourced_str(b, _LLM_SOURCE, Confidence.HIGH) for b in d.other_boards]`
         - `is_overboarded`: `(len(d.other_boards) + 1) >= 4`
       - Skip directors with empty name
       - Return list of profiles

    2. `convert_board_profile(extraction: DEF14AExtraction) -> BoardProfile`
       - `size`: from `extraction.board_size`
       - `independence_ratio`: compute from `extraction.independent_count / extraction.board_size` if both present
       - `avg_tenure_years`: compute mean of `[d.tenure_years for d in extraction.directors if d.tenure_years is not None]`
       - `ceo_chair_duality`: from `extraction.ceo_chair_combined`
       - `overboarded_count`: count directors with 4+ other_boards
       - `classified_board`: from `extraction.classified_board`
       - All wrapped in sourced_int/sourced_float/SourcedValue[bool] with _LLM_SOURCE

    3. `convert_compensation(extraction: DEF14AExtraction) -> CompensationAnalysis`
       - Find the CEO among `extraction.named_executive_officers` (title contains "CEO" or "Chief Executive")
       - If CEO found: populate `ceo_salary`, `ceo_bonus`, `ceo_equity` (stock_awards + option_awards), `ceo_other`, `ceo_total_comp` using sourced_float
       - `ceo_pay_ratio`: parse "123:1" string to float 123.0
       - `say_on_pay_pct`: from `extraction.say_on_pay_approval_pct`
       - `has_clawback`: from `extraction.has_clawback` (new field from 19-01)
       - `clawback_scope`: from `extraction.clawback_scope` (new field from 19-01)
       - `related_party_transactions`: not in DEF14A schema -- leave empty (comes from 10-K)

    4. `convert_compensation_flags(extraction: DEF14AExtraction) -> CompensationFlags`
       - `say_on_pay_support_pct`: from `extraction.say_on_pay_approval_pct`
       - `ceo_pay_ratio`: parse "123:1" to float
       - `golden_parachute_value`: from `extraction.golden_parachute_total`

    5. `convert_ownership_from_proxy(extraction: DEF14AExtraction) -> OwnershipAnalysis`
       - `insider_pct`: from `extraction.officers_directors_ownership_pct`
       - Only partially populates ownership (institutional_pct comes from yfinance, activist data from 13D/G extractors)
       - Return OwnershipAnalysis with just insider_pct set

    6. `convert_neos_to_leaders(extraction: DEF14AExtraction) -> list[LeadershipForensicProfile]`
       - Convert each NEO to a LeadershipForensicProfile with name, title, bio_summary (from source_passage)
       - These supplement (not replace) the leadership profiles from the leadership extractor

    7. `_parse_pay_ratio(ratio_str: str | None) -> float | None`
       - Parse "123:1" or "123 to 1" or "123" to float 123.0
       - Return None on parse failure

    **Import pattern:** Use `from do_uw.stages.extract.sourced import sourced_str, sourced_float, sourced_int, now` for SourcedValue construction. Use `from do_uw.models.common import Confidence, SourcedValue` for the types.

    **Pyright strict:** All functions must have complete type annotations. Use `from __future__ import annotations`. `SourcedValue[bool]` constructor needs explicit `as_of=now()`.
  </action>
