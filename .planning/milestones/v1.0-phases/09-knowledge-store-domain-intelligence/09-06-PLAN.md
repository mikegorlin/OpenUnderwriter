    2. Create `src/do_uw/knowledge/playbooks.py` (under 500 lines):

    **load_playbooks(store: KnowledgeStore) -> int**:
    - Load all INDUSTRY_PLAYBOOKS into knowledge store IndustryPlaybook table
    - Skip if already loaded (check by ID)
    - For each playbook's industry_checks, create Check entries with status=INCUBATING, origin=BRAIN_MIGRATION, section inferred from check type
    - Return count of playbooks loaded

    **activate_playbook(sic_code: str, naics_code: str | None, store: KnowledgeStore) -> dict[str, Any] | None**:
    - Parse SIC code to int (first 4 digits)
    - Check each playbook's sic_ranges -- if SIC falls in any range, return that playbook
    - If no SIC match and naics_code provided, check naics_prefixes
    - Return playbook dict with all fields, or None if no match

    **get_industry_checks(store: KnowledgeStore, playbook_id: str) -> list[dict[str, Any]]**:
    - Query checks associated with this playbook (via metadata_json containing playbook_id)
    - Return list of check dicts

    **get_industry_questions(store: KnowledgeStore, playbook_id: str) -> list[str]**:
    - Get meeting prep questions from playbook
    - Return list of question strings

    **get_scoring_adjustments(store: KnowledgeStore, playbook_id: str) -> dict[str, float]**:
    - Get factor-level scoring weight adjustments
    - Return dict of factor_id -> multiplier

    **get_claim_theories(store: KnowledgeStore, playbook_id: str) -> list[str]**:
    - Get industry-specific claim theories
    - Return list of theory strings

    **get_active_checks_with_industry(store: KnowledgeStore, playbook_id: str | None) -> list[dict[str, Any]]**:
    - Returns ALL active checks: generic (status=ACTIVE) + industry-specific (if playbook_id provided)
    - If playbook_id is None, returns only generic ACTIVE checks (same as current behavior)
    - If playbook_id is set, ALSO includes INCUBATING checks tagged with that playbook_id in metadata_json
    - This is the single entry point for BackwardCompatLoader to return the unified check list
    - Industry checks are appended AFTER generic checks so they don't affect existing check ordering
