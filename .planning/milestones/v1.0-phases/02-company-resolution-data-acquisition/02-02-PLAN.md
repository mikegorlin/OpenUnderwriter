           d. For each filing type, use `FallbackChain` with tiers:

           e. Filing lookback: 3 years for annual reports, trailing quarters, 3 proxy statements
           f. Cache each filing response with appropriate TTL from CACHE_TTLS
           g. Return dict keyed by filing type: `{"10-K": [...], "10-Q": [...], "DEF 14A": [...], ...}`
           h. Each filing entry is a dict with: accession_number, filing_date, form_type, primary_doc_url, filing_url
           i. Do NOT download full filing content yet -- just the filing metadata and URLs. Full content download happens during EXTRACT stage or can be fetched on demand.
       - `CACHE_TTLS` dict with per-type TTLs (from research): 10-K/20-F = 14 months, 10-Q/6-K = 5 months, DEF 14A = 14 months, 8-K = 30 days, Form 4 = 7 days
       - `FPI_FILING_MAP` dict mapping generic types to FPI equivalents: `{"annual": ["10-K", "20-F"], "quarterly": ["10-Q", "6-K"], ...}`

       - `WebSearchClient` class:

         - `search(self, query: str, cache: AnalysisCache | None = None) -> list[dict[str, str]]`: Perform a single web search.
           - Uses Claude's built-in WebSearch tool if available, or Brave Search MCP, or falls back to an empty result with a warning.

           - Track search count: increment counter, check against budget, log warning at 80% budget.
           - Store search count in cache: key `brave:count:{YYYY-MM}`, increment on each search.
           - Check monthly limit: warn if approaching 2,000.
         - `blind_spot_sweep(self, company_name: str, ticker: str, cache: AnalysisCache | None = None) -> dict[str, list[dict]]`: Run the full blind spot discovery battery.

             4. `"{company_name}" restatement whistleblower scandal`
             5. `"{company_name}" FDA warning CFPB OSHA` (industry-specific, skip if budget tight)
           - Runs searches in priority order, stops if budget exhausted.
           - Returns dict keyed by search category: `{"litigation": [...], "regulatory": [...], "short_seller": [...], "whistleblower": [...], ...}`
         - Cache results with 30-day TTL.
