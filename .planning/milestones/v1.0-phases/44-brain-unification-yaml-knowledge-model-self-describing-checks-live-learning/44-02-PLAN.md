This script reads checks.json, patterns.json, red_flags.json, and causal_chains.yaml, then outputs domain YAML files under src/do_uw/brain/checks/.

**Script structure (all functions, not monolithic):**

```python
# Top-level functions:
def load_sources() -> tuple[list[dict], list[dict], list[dict], dict]:
    """Load checks.json, patterns.json, red_flags.json, causal_chains.yaml"""

def build_chain_roles_index(causal_chains: dict) -> dict[str, dict[str, str]]:
    """Returns {check_id: {chain_id: role}} for all 117 cross-referenced checks.
    Parse causal_chains.yaml steps — each step has check_id and role field."""

def build_red_flag_index(red_flags: list[dict]) -> set[str]:
    """Returns set of check IDs that appear as source_check in any red_flag entry."""

def map_content_type_to_work_type(content_type: str) -> str:
    """MANAGEMENT_DISPLAY→extract, EVALUATIVE_CHECK→evaluate, INFERENCE_PATTERN→infer"""

def derive_acquisition_tier(required_data: list[str]) -> str:
    """L1: SEC_10K/10Q/8K/XBRL_*; L2: item_* text; L3: MARKET_PRICE/NEWS/COURT/WEB; L4: computed"""

def map_hazard_or_signal_to_layer(hazard_or_signal: str) -> str:
    """HAZARD→hazard, SIGNAL→signal, PERIL_CONFIRMING→peril_confirming"""

def derive_worksheet_section(section_int: int, check_id: str) -> str:
    """Map old int section (1-7) to semantic string using SECTION_MAP dict"""

def migrate_single_check(
    check: dict,
    chain_roles_index: dict,
    red_flag_ids: set,
) -> dict:
    """Transform one checks.json entry to unified YAML dict.
    Adds: work_type, layer, acquisition_tier, worksheet_section, display_when,
          chain_roles, peril_ids (empty list), unlinked, provenance.
    Removes: pillar, category, signal_type, hazard_or_signal, content_type, section.
    Keeps old fields as-is: id, name, factors, required_data, data_locations, threshold,
          execution_mode, claims_correlation, tier, depth, pattern_ref, plaintiff_lenses,
          v6_subsection_ids, amplifier, amplifier_bonus_points, sector_adjustments,
          extraction_hints, data_strategy."""

def assign_domain_file(check_id: str) -> str:
    """Returns relative path within checks/ dir.
    GOV.BOARD.* → gov/board.yaml, GOV.AUDIT.* → gov/audit.yaml
    GOV.EXEC.* → gov/exec_comp.yaml, GOV.ACTIVIST.* → gov/activist.yaml
    GOV.PAY.* → gov/pay.yaml, GOV.RIGHTS.* → gov/rights.yaml
    GOV.* (other) → gov/effect.yaml
    FWRD.GUID.* or FWRD.FORECAST.* → fwrd/guidance.yaml
    FWRD.SPAC.* or FWRD.MERGER.* → fwrd/spac.yaml
    FWRD.MA.* or FWRD.ACQ.* → fwrd/ma.yaml
    FWRD.* (other) → fwrd/transform.yaml
    BIZ.* → biz.yaml, FIN.* → fin.yaml, EXEC.* → exec.yaml
    LIT.* → lit.yaml, STOCK.* → stock.yaml, NLP.* → nlp.yaml"""

def write_domain_yaml_files(grouped: dict[str, list[dict]], output_dir: Path) -> None:
    """Write each domain's checks to YAML file. Create subdirs as needed.
    Use ruamel.yaml or PyYAML — check which is available in pyproject.toml."""

def verify_output(output_dir: Path, original_count: int) -> None:
    """Count total checks in all YAML files. Assert == original_count.
    Count files that exceed 500 lines and warn."""

def main() -> None:
    """Orchestrate: load → build indexes → migrate → group → write → verify"""
```

**Key transformation rules:**
- provenance for all migrated checks: `{origin: migrated_from_json, confidence: inherited, last_validated: null, source_url: null, source_date: null, source_author: null, added_by: null}`
- chain_roles: use chain_roles_index lookup — if check_id found → use the dict, else → `{}`
- unlinked: True if chain_roles == {}, False otherwise
- peril_ids: empty list `[]` for all migrated checks (populated in a future phase via causal chain traversal — see RESEARCH.md pitfall note)
- display_when: default to "always" for extract, "has_data" for evaluate, "fired" for infer
- critical_red_flag: True if check_id in red_flag_ids set

**YAML formatting:** Write each check as a YAML mapping with a blank line between entries. Use block style (not flow) for nested dicts and lists. Include a header comment in each file: `# Generated by brain_migrate_yaml.py — DO NOT EDIT — source: checks.json`

**500-line limit compliance:** After writing, call verify_output which counts lines per file. Any file over 490 lines triggers a warning with suggested split point (by sub-prefix of ID).
