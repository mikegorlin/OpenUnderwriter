**brain_migrate.py changes:**

Read the file first. Find where it currently loads checks.json (likely something like `json.load(open('brain/checks.json'))`). Replace that logic with a YAML glob loader:

```python
import glob
import yaml
from pathlib import Path

def load_checks_from_yaml(checks_dir: Path) -> list[dict]:
    """Load all checks from checks/**/*.yaml glob. Returns flat list of check dicts."""
    all_checks = []
    pattern = str(checks_dir / "**" / "*.yaml")
    for yaml_file in sorted(glob.glob(pattern, recursive=True)):
        data = yaml.safe_load(Path(yaml_file).read_text())
        if isinstance(data, list):
            all_checks.extend(data)
        elif isinstance(data, dict) and "checks" in data:
            all_checks.extend(data["checks"])
    return all_checks
```
