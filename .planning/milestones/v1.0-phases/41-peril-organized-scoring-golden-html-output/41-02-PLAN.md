**Step 1: Replace single-quarter rendering with multi-quarter loop.**
In `financial.html.j2`, replace the current single-quarter block (around line 200-240) with a loop over ALL quarterly updates. The current code is:
```jinja2
{% set qu_list = fin.get('quarterly_updates', []) %}
{% if qu_list %}
{% set qu = qu_list[0] %}
<h3>Post-Annual Update: {{ qu.quarter }}</h3>
...
{% endif %}
```

Replace with:
```jinja2
{% set qu_list = fin.get('quarterly_updates', []) %}
{# Filter to quarters with at least one non-N/A financial metric #}
{% set qu_valid = [] %}
{% for qu in qu_list %}
  {% if qu.revenue != 'N/A' or qu.net_income != 'N/A' or qu.eps != 'N/A' %}
    {% set _ = qu_valid.append(qu) %}
  {% endif %}
{% endfor %}
{% if qu_valid %}
<h3>Quarterly Updates ({{ qu_valid|length }} Quarter{{ 's' if qu_valid|length > 1 }})</h3>
{% for qu in qu_valid %}
<div class="{% if not loop.first %}mt-4 pt-3 border-t border-gray-200{% endif %} page-break-inside-avoid">
  <h4>{{ 'Most Recent' if loop.first else 'Prior' }}: {{ qu.quarter }}</h4>
  ...existing table rendering for this quarter...
</div>
{% endfor %}
```

The first quarter gets "Most Recent" label, subsequent get "Prior" label. Each quarter renders in its own bordered div.

**IMPORTANT:** Keep ALL existing per-quarter content (financial table with revenue/net_income/eps, MD&A highlights, new legal proceedings, going concern, material weaknesses, subsequent events). Just wrap in a `{% for qu in qu_valid %}` loop.

**Step 3: Handle Jinja2 list append pattern.**
Jinja2 does not support `list.append()` directly. Use a namespace or restructure. The cleanest approach: use `selectattr` or a Jinja2-compatible filter. Alternatively, use `{% set ns = namespace(valid=[]) %}` with `{% set ns.valid = ns.valid + [qu] %}` pattern:
```jinja2
{% set ns_qu = namespace(valid=[]) %}
{% for qu in qu_list %}
  {% if qu.revenue != 'N/A' or qu.net_income != 'N/A' or qu.eps != 'N/A' %}
    {% set ns_qu.valid = ns_qu.valid + [qu] %}
  {% endif %}
{% endfor %}
{% set qu_valid = ns_qu.valid %}
```
