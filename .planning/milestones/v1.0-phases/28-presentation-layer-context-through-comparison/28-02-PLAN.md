  <action>
  Create src/do_uw/stages/render/peer_context.py with these functions:

  1. `format_metric_with_context(label: str, value: str, benchmark: MetricBenchmark | None, named_peers: list[str] | None = None) -> str`
     - If benchmark is None or percentile_rank is None, return f"{label}: {value}"
     - Otherwise: "{label}: {value} ({Nth} percentile vs. {peer_count} peers; median {baseline_value})"
     - If named_peers provided, append " [{PEER1}, {PEER2}, {PEER3}]" (max 3)
     - Use ordinal suffix (1st, 2nd, 3rd, 4th, ..., 92nd, etc.)
     - Higher percentile = higher relative value; let caller decide if that's good or bad

  2. `get_peer_context_line(metric_key: str, benchmark_result: BenchmarkResult | None) -> str | None`
     - Looks up metric_key in benchmark_result.metric_benchmarks (if that field exists) or peer_rankings
     - Returns a formatted sentence: "Ranks at the {N}th percentile among {count} peers (peer median: {baseline})"
     - Returns None if data unavailable

  3. `get_benchmark_for_metric(metric_key: str, state: AnalysisState) -> MetricBenchmark | None`
     - Safely extracts a MetricBenchmark from state.benchmarked.metric_benchmarks if available
     - Returns None if state.benchmarked is None or metric not found

  4. `render_peer_comparison_narrative(doc: Any, state: AnalysisState, ds: DesignSystem) -> None`
     - Renders a "Peer Group Comparison" subsection using doc.add_paragraph()
     - Shows named peers from state.benchmarked.peer_group_tickers with PeerCompany details
     - For each named peer: ticker, name, market cap, and peer_quality_score if available
     - Narrative sentence: "Among [company]'s [N] closest peers, the company ranks at the [X]th percentile by quality score."
     - If peer_quality_scores available, show top 3 closest peers with scores
     - Uses add_styled_table() for the peer comparison table

  5. `_ordinal(n: int) -> str` -- helper: 1->"1st", 2->"2nd", 3->"3rd", 4->"4th", 11->"11th", 21->"21st"

  In formatters.py, add `format_percentile(rank: float) -> str` that returns e.g. "92nd percentile" using the ordinal helper. Keep formatters.py under 500 lines.

  Create tests/stages/render/test_peer_context.py with tests for:
  - format_metric_with_context with None benchmark (plain format)
  - format_metric_with_context with full benchmark data
  - format_metric_with_context with named peers
  - _ordinal edge cases (1st, 2nd, 3rd, 11th, 12th, 13th, 21st, 22nd, 23rd, 111th, 112th)
  - get_peer_context_line with missing data returns None
  - get_benchmark_for_metric with None state.benchmarked returns None
  </action>
