  **Main function:**
  ```python
  def extract_debt_analysis(
      state: AnalysisState,
  ) -> tuple[
      SourcedValue[dict[str, float | None]] | None,   # liquidity
      SourcedValue[dict[str, float | None]] | None,   # leverage
      SourcedValue[dict[str, Any]] | None,             # debt_structure
      SourcedValue[dict[str, Any]] | None,             # refinancing_risk
      list[ExtractionReport],
  ]:
  ```

  **1. Liquidity Assessment** (SECT3-08):
  Extract from balance sheet:
  - Current Ratio = Current Assets / Current Liabilities
  - Quick Ratio = (Current Assets - Inventory) / Current Liabilities
  - Cash Ratio = Cash & Equivalents / Current Liabilities
  - Working Capital = Current Assets - Current Liabilities
  - Days Cash on Hand = Cash / (Operating Expenses / 365)

  Read values from `state.extracted.financials.statements.balance_sheet` FinancialStatement model. Navigate line_items by label or xbrl_concept.

  Helper function to find a line item value:
  ```python
  def _get_line_item_value(statement: FinancialStatement, concept: str, period: str) -> float | None:
      for item in statement.line_items:
          if item.xbrl_concept == concept:
              sv = item.values.get(period)
              return sv.value if sv is not None else None
      return None
  ```

  Compute for latest period. If prior period available, include both for trend.
  Return as SourcedValue with source="Derived from XBRL balance sheet" and confidence=HIGH (inputs from audited statements).

  **2. Leverage Assessment** (SECT3-09):
  Compute from income statement + balance sheet:
  - Debt-to-Equity = Total Debt / Stockholders' Equity
  - Debt-to-EBITDA = Total Debt / EBITDA (where EBITDA = Operating Income + D&A)
  - Interest Coverage = EBIT / Interest Expense (handle zero interest expense -> "No debt service")
  - Debt-to-Assets = Total Debt / Total Assets
  - Net Debt = Total Debt - Cash & Equivalents

  Flag if:
  - Debt-to-EBITDA > 4.0 (high leverage)
  - Interest Coverage < 2.0 (thin coverage)
  - Debt-to-Equity > 3.0 (aggressive capital structure)

  **3. Debt Structure** (SECT3-10):
  Extract from 10-K text (Item 7 MD&A or footnotes):
  - Maturity schedule: Search for debt maturity table in filing text. Look for patterns like "Due in {year}" or "Matures {date}". Parse into {year: amount} dict.
  - Interest rates: Search for "interest rate", "coupon", "floating rate", "SOFR", "LIBOR" in text. Classify as fixed vs. floating.
  - Covenant information: Search for "covenant", "debt covenant", "financial covenant" in text. Extract any mentioned ratios or conditions.
  - Credit facility details: Search for "revolving credit", "credit agreement", "credit facility" and extract facility size, drawn amount, availability.

  This is text-based extraction from filing HTML/text. Results are MEDIUM confidence (extracted from official filings but parsing may be imprecise). When filing text is not available in acquired_data, use `sec_get_text()` to fetch (mark as ACQUIRE expansion).

  If text-based parsing fails to find structured data, return "Not Available" with reason="parsing_incomplete" rather than fabricating a maturity schedule.

  **4. Refinancing Risk** (SECT3-11):
  Compute from debt structure + liquidity:
  - Near-term maturities: Sum of debt maturing within 2 years
  - Maturity wall: Largest single-year maturity amount
  - Coverage: Available cash + credit facility vs. near-term maturities
  - Refinancing risk level: LOW (ample coverage), MEDIUM (tight but manageable), HIGH (shortfall), CRITICAL (imminent maturity with no coverage)

  This requires debt_structure output. If debt_structure is "Not Available", refinancing_risk is also "Not Available" with reason="depends_on_debt_structure".

  **ExtractionReport for each sub-module:**
  - Liquidity: expected=[current_ratio, quick_ratio, cash_ratio, working_capital, days_cash], found=whichever computed
  - Leverage: expected=[d/e, d/ebitda, interest_coverage, d/a, net_debt], found=whichever computed
  - Debt structure: expected=[maturity_schedule, rates, covenants, credit_facility], found=whichever parsed
  - Refinancing: expected=[near_term_maturities, maturity_wall, coverage, risk_level], found=whichever computed
