**1. `src/do_uw/knowledge/pricing_models.py`** -- SQLAlchemy ORM models for pricing data. Import `Base` from `do_uw.knowledge.models`. Follow the Mapped[] annotation pattern from models.py for pyright strict compliance.

Create `Quote(Base)` table ("quotes"):
- id: Mapped[int] (Integer PK, autoincrement)
- ticker: Mapped[str] (String, not null, indexed)
- company_name: Mapped[str] (String, not null)
- effective_date: Mapped[datetime] (DateTime, not null, indexed)
- expiration_date: Mapped[datetime | None] (DateTime, nullable)
- quote_date: Mapped[datetime] (DateTime, not null)
- status: Mapped[str] (String, not null, default "QUOTED") -- values: INDICATION, QUOTED, BOUND, EXPIRED, DECLINED
- total_limit: Mapped[float] (Float, not null) -- USD
- total_premium: Mapped[float] (Float, not null) -- USD
- retention: Mapped[float | None] (Float, nullable) -- SIR/deductible USD
- market_cap_tier: Mapped[str] (String, not null, indexed) -- MEGA/LARGE/MID/SMALL/MICRO
- sic_code: Mapped[str | None] (String, nullable)
- sector: Mapped[str | None] (String, nullable, indexed)
- quality_score: Mapped[float | None] (Float, nullable) -- system quality score at quote time
- tier: Mapped[str | None] (String, nullable) -- system tier at quote time
- program_rate_on_line: Mapped[float] (Float, not null) -- computed: total_premium / total_limit
- source: Mapped[str] (String, not null) -- who entered this data
- notes_text: Mapped[str | None] (String, nullable) -- use notes_text to avoid clash with relationship name
- created_at: Mapped[datetime] (DateTime, not null)
- metadata_json: Mapped[str | None] (String, nullable)
- layers: Mapped[list[TowerLayer]] relationship (back_populates="quote", cascade="all, delete-orphan")

Create `TowerLayer(Base)` table ("tower_layers"):
- id: Mapped[int] (Integer PK, autoincrement)
- quote_id: Mapped[int] (Integer FK to quotes.id, not null)
- layer_position: Mapped[str] (String, not null) -- PRIMARY/LOW_EXCESS/MID_EXCESS/HIGH_EXCESS
- layer_number: Mapped[int] (Integer, not null) -- 1-based
- attachment_point: Mapped[float] (Float, not null) -- USD (0 for primary)
- limit_amount: Mapped[float] (Float, not null) -- USD per layer (use limit_amount to avoid shadowing Python built-in)
- premium: Mapped[float] (Float, not null) -- USD per layer
- carrier_name: Mapped[str] (String, not null)
- carrier_rating: Mapped[str | None] (String, nullable) -- AM Best
- rate_on_line: Mapped[float] (Float, not null) -- computed: premium / limit_amount
- premium_per_million: Mapped[float] (Float, not null) -- computed: premium / (limit_amount / 1e6)
- is_lead: Mapped[bool] (Integer as bool, not null, default False)
- share_pct: Mapped[float | None] (Float, nullable) -- quota share percentage
- quote: Mapped[Quote] relationship (back_populates="layers")

Add `__repr__` methods to both classes. Keep file under 150 lines.

**1. `src/do_uw/knowledge/pricing_store.py`** -- PricingStore class. Follow KnowledgeStore pattern (contextmanager _session, get_session, engine creation).

Constructor: `__init__(self, db_path: str | Path | None = _DEFAULT_DB_PATH)` using same default path as KnowledgeStore (knowledge.db in package dir). Create engine, session factory, call `Base.metadata.create_all(self._engine)` to ensure pricing tables exist.

Methods:
- `add_quote(self, quote_input: QuoteInput) -> int`: Create Quote ORM object from Pydantic input. Compute program_rate_on_line = total_premium / total_limit (guard against zero). Set created_at to now(UTC). For each layer in quote_input.layers, create TowerLayer ORM with computed rate_on_line = premium / limit_amount and premium_per_million = premium / (limit_amount / 1e6). Guard against zero division on both. session.add(quote), session.flush(), return quote.id.
- `get_quote(self, quote_id: int) -> QuoteOutput | None`: Get quote by ID with eager-loaded layers. Return as QuoteOutput Pydantic model, or None.
- `list_quotes(self, ticker: str | None = None, status: str | None = None, limit: int = 50) -> list[QuoteOutput]`: Filter quotes by ticker and/or status, ordered by effective_date desc.
- `add_tower_layer(self, quote_id: int, layer: TowerLayerInput) -> int`: Add a layer to an existing quote. Compute rate_on_line and premium_per_million. Return layer.id.
- `get_tower_comparison(self, ticker: str, limit: int = 5) -> list[dict[str, Any]]`: Return most recent tower structures for a company (dict format as shown in research).
- `update_quote_status(self, quote_id: int, new_status: str) -> bool`: Update status field. Return True if quote found and updated.
- `get_rates_for_segment(self, market_cap_tier: str | None = None, sector: str | None = None, layer_position: str | None = None, status_filter: list[str] | None = None, months_back: int = 24) -> list[float]`: Return rate_on_line values for quotes matching the segment filters within the time window. Default status_filter = ["QUOTED", "BOUND"]. This is the primary data method for the analytics engine (Plan 10-02).

Use `from __future__ import annotations` at top. Import Quote and TowerLayer from pricing_models. Keep under 400 lines.
