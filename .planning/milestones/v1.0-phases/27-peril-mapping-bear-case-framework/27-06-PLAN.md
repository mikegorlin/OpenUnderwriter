Create `src/do_uw/stages/score/frequency_model.py` with a single public function `compute_enhanced_frequency()`.

The function signature:

```python
def compute_enhanced_frequency(
    state: AnalysisState,
    red_flag_results: list[RedFlagResult],
    patterns: list[PatternMatch],
    factor_scores: list[FactorScore],
) -> EnhancedFrequency:
```

**Formula:** `adjusted_probability = base_rate * hazard_mult * signal_mult`

Where:
- `base_rate` = `state.classification.base_filing_rate_pct` (from Phase 25 classification engine). If classification is None, fall back to `claim_prob.range_high_pct` from existing tier-based computation (backward compat).
- `hazard_mult` = `state.hazard_profile.ies_multiplier` (from Phase 25 hazard profile). Default 1.0 if hazard_profile is None.
- `signal_mult` = computed from 3 signal sources:
  1. **CRF trigger count**: Each triggered CRF gate adds to signal. 0 triggers = 1.0x, 1 trigger = 1.15x, 2 = 1.30x, 3+ = 1.50x. Read multiplier values from a `_SIGNAL_CONFIG` dict at module level (not hardcoded inline).
  2. **Active pattern count**: Each detected pattern (EVENT_COLLAPSE, DEATH_SPIRAL, etc.) adds. 0 = 1.0x, 1-2 = 1.10x, 3+ = 1.25x.
  3. **Elevated factor ratio**: Ratio of factors with points_deducted > 50% of max_points. If >50% of factors are elevated, 1.15x. If >75%, 1.30x. Otherwise 1.0x.

  Final `signal_mult = crf_mult * pattern_mult * factor_mult`, capped at 2.0x to prevent runaway.

**Output model** `EnhancedFrequency` (Pydantic BaseModel, define in same file to keep it self-contained):
- `base_rate_pct: float` -- classification base filing rate
- `hazard_multiplier: float` -- IES multiplier
- `signal_multiplier: float` -- combined signal multiplier
- `crf_signal: float` -- CRF contribution
- `pattern_signal: float` -- pattern contribution
- `factor_signal: float` -- factor elevation contribution
- `adjusted_probability_pct: float` -- final result (capped at 50%)
- `methodology: str` -- "classification x hazard x signal" or "tier-based fallback"
- `components: dict[str, float]` -- breakdown for transparency

Cap the final `adjusted_probability_pct` at 50.0% (no company has >50% annual filing probability).

- Test with patterns elevating signal

- Test combined signals capped at 2.0x
- Test fallback when classification is None
- Test final probability capped at 50%
- Test that 0 CRF / 0 patterns / 0 elevated = signal_mult of 1.0x (clean company)
