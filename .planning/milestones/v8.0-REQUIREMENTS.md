# Requirements: D&O Underwriting System — v8.0 Intelligence Dossier

**Defined:** 2026-03-18
**Core Value:** Transform every data point into D&O risk intelligence so underwriters see what the data MEANS, not just what the data IS.
**Gold Standard Reference:** Feedme/HNGE - D&O Analysis - 2026-03-18.pdf (29 pages)

## v8.0 Requirements

### Narrative Quality (QUAL) — Cross-Cutting

These apply to EVERY piece of generated text in the worksheet. No exceptions.

- [ ] **QUAL-01**: Every key negative/positive MUST lead with specific data (dollar amounts, percentages, dates, names) — never generic statements like "faces risk from critical red flag" or "has experienced a notable decline"
- [ ] **QUAL-02**: Every key negative/positive MUST reference the scoring factor it maps to (e.g., "F.4 = 8/10" or "F.7 Insider Trading: 5/8")
- [ ] **QUAL-03**: Every LLM extraction prompt MUST include the full analytical context: scoring results, signal evaluations, financial data, and company-specific details — not just the filing text
- [ ] **QUAL-04**: Zero boilerplate/template language in any underwriter-facing narrative. Every sentence must contain company-specific information that could ONLY apply to this company. If a sentence could apply to any company by changing the name, it fails.
- [ ] **QUAL-05**: Key negatives must include: the specific finding, the dollar/percentage magnitude, the scoring factor reference, and the D&O litigation theory it maps to. Example: "Heavy insider selling: $193M / 5.5% of market cap — Chairman Mecklenburg $7.5M (Mar 2026), CEO Perez $6.7M (Feb 2026). F.7 = 5/8. Plaintiff attorneys cite insider selling + stock decline as circumstantial evidence of scienter."
- [ ] **QUAL-06**: Key positives must include: the specific evidence, quantification, and WHY it reduces D&O risk. Example: "Zero litigation history — F.1 = 0/20. Stanford SCAC clean, no SEC enforcement, no derivative suits. The 20-point F.1 factor is the heaviest weight in the scoring system."
- [ ] **QUAL-07**: All section-opening narratives must be company-specific analytical paragraphs that tell the STORY of this company's risk profile, not generic descriptions of what the section contains

- [x] **INFRA-02**: do_context engine in ANALYZE evaluates templates against signal results, producing rendered commentary strings stored on signal result

- [x] **INFRA-05**: Templates render do_context strings as-is — zero D&O interpretation in Jinja2

### Company Intelligence Dossier (DOSSIER)

- [x] **DOSSIER-01**: "What the Company Does" subsection: plain English business description + "The core D&O exposure" paragraph identifying single most important risk vector (LLM extraction from 10-K + S-1)
- [x] **DOSSIER-02**: "How Money Flows" subsection: revenue flow diagram showing business model visually (employer→contract→delivery→renewal pattern)
- [x] **DOSSIER-03**: "Revenue Model Card" table: model type, pricing, revenue quality, recognition method, contract duration, NDR, retention, ARPU, rev-rec complexity, guidance miss risk, concentration risk, regulatory overlay — each with D&O Risk column
- [x] **DOSSIER-04**: "Revenue Segment Breakdown" with D&O Litigation Exposure column per segment + multi-dimensional concentration assessment (customer, geographic, product, channel, payer) with risk levels and D&O implications
- [x] **DOSSIER-05**: "Unit Economics" table: key business metrics with Value/Benchmark/Assessment columns + narrative identifying the SINGLE MOST IMPORTANT METRIC for this company's D&O risk
- [x] **DOSSIER-06**: "Revenue Waterfall (Growth Decomposition)": starting→new→expansion→churn→ending for key business metrics with D&O insight narrative
- [x] **DOSSIER-07**: "Competitive Landscape & Moat Assessment": peer comparison table (4+ companies, 8+ dimensions) + moat assessment (data advantage, switching costs, scale, brand, network effects, regulatory barrier, distribution lock) with Present/Strength/Durability
- [x] **DOSSIER-08**: "Emerging Risk Radar": risk/probability/impact/timeframe/D&O Factor/status table for next 12-18 months
- [x] **DOSSIER-09**: "Revenue Recognition (ASC 606)": element/approach/complexity/D&O Risk table from 10-K footnotes + billings vs revenue gap analysis

### D&O Commentary Layer (COMMENT)

- [x] **COMMENT-01**: Every data table in the worksheet has an optional "D&O Risk" or "D&O Implication" or "Assessment" column populated by signal do_context
- [x] **COMMENT-02**: Each scoring factor (F.1-F.10) has "What Was Found" section with data citations and "Underwriting Commentary" section explaining WHY the score matters in context
- [x] **COMMENT-03**: Each forensic indicator (Beneish, Sloan, DSO, etc.) has D&O Commentary explaining relevance to D&O risk profile
- [x] **COMMENT-04**: Financial Health section opens with narrative paragraph explaining the financial STORY before showing data tables
- [x] **COMMENT-05**: Governance section opens with narrative explaining governance posture and D&O implications
- [x] **COMMENT-06**: Litigation section narrative contextualizes clean/active docket with sector comparison and SCA probability

### Stock & Market Intelligence (STOCK)

- [x] **STOCK-01**: Every stock drop >5% has "Catalyst & D&O Assessment" column with full catalyst identification (earnings, 8-K, market-wide, analyst action) + D&O litigation risk assessment
- [x] **STOCK-02**: Stock drops table includes From price, To price, Volume alongside drop percentage
- [x] **STOCK-03**: "D&O Underwriting Implication" narrative after stock drop table explaining overall drop pattern significance (catalytic vs gradual, corrective disclosure vs market-driven)
- [x] **STOCK-04**: Stock performance summary with multi-horizon returns (1D, 5D, 1M, 3M, 6M, Since IPO, 52W Decline) and contextual notes
- [x] **STOCK-05**: "Pattern Detection" section identifying notable patterns (post-IPO arc, multi-day clusters, lockup expiry impact, support levels)
- [x] **STOCK-06**: Analyst consensus table (rating distribution, mean target, high/low, coverage count) with interpretive narrative

### Forward-Looking Intelligence (FORWARD)

- [x] **FORWARD-01**: "Forward-Looking Statement Risk Map": every publicly guided metric mapped to current value → guidance/claim → miss risk → SCA relevance if missed
- [x] **FORWARD-02**: "Management Credibility" table: quarter-by-quarter guidance vs actual with beat/miss/magnitude, track record percentage, credibility score
- [x] **FORWARD-03**: "2026 Catalysts" table: event/timing/impact if negative/litigation risk for each forward catalyst
- [x] **FORWARD-04**: "Monitoring Triggers" table: trigger/action/threshold for ongoing post-bind monitoring (SCA filing, stock below support, insider selling pace, CEO departure, earnings miss, yield regression, peer SCA)
- [x] **FORWARD-05**: "Growth Estimates" table: current quarter, current year, next year EPS estimates with trend assessment
- [x] **FORWARD-06**: "Alternative Forward-Looking Signals" section: short interest trend, analyst sentiment shifts, buyback support, meeting notes context

### Scoring & Recommendation (SCORE)

- [x] **SCORE-01**: Scoring detail section: each factor (F.1-F.10) expanded with "What Was Found" (data with sources) + "Underwriting Commentary" (contextual D&O interpretation)
- [x] **SCORE-02**: "ZER-001 Verifications" section: explicit verification of zero-scored factors (F.1=0, F.3=0, F.9=0) with positive evidence citations
- [x] **SCORE-03**: "Suggested Underwriting Posture" table: decision/retention/limit/pricing/exclusions/monitoring/re-evaluation with rationale per element
- [x] **SCORE-04**: "Why [TIER], not [ADJACENT_TIER]" narrative explaining tier placement with specific factor references
- [x] **SCORE-05**: "Watch Items" section: specific items requiring ongoing attention with clear thresholds for re-evaluation

### Trigger & Quick Screen (TRIGGER)

- [x] **TRIGGER-01**: "Trigger Matrix Summary" section: all RED/YELLOW flags from quick screen with flag level, deep-dive section routing, and routed checkmark
- [x] **TRIGGER-02**: "Prospective Checks" table: forward-looking checks (earnings expectations, major contract/deal, regulatory decision, competitive disruption, macro headwinds) with findings and traffic light status
- [x] **TRIGGER-03**: Nuclear triggers checked and reported (0 triggered = clean)

### Alternative Data (ALTDATA)

- [x] **ALTDATA-01**: "ESG/Greenwashing Risk" assessment: ESG controversies, mission alignment, MSCI/ISS ratings, greenwashing indicators with D&O Relevance
- [x] **ALTDATA-02**: "AI-Washing Risk" assessment: AI claim indicators (present/evidence/D&O risk) with scienter risk assessment
- [x] **ALTDATA-03**: "Tariff/Trade Exposure" assessment: supply chain, international ops, manufacturing exposure with D&O Relevance
- [x] **ALTDATA-04**: "Competitor SCA Check": active SCAs against direct competitors flagged as sector contagion risk

## Out of Scope

| Feature | Reason |
|---------|--------|
| Earnings call transcript API integration | Paid service violates public-data-only constraint; 8-K Item 2.02 + yfinance covers 80%+ |
| LLM-at-render-time commentary | Non-deterministic, non-auditable, breaks brain portability |
| Real-time monitoring dashboard | Different product; worksheet is point-in-time analysis |
| Automated bind/decline decisions | System augments underwriter judgment, doesn't replace it |
