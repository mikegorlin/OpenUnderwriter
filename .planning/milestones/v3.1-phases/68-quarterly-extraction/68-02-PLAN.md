  <action>
    Create src/do_uw/stages/extract/xbrl_trends.py (~200 lines):

    1. `TrendResult` dataclass with fields: concept (str), qoq_changes (list[float | None]), yoy_changes (list[float | None]), acceleration (float | None), consecutive_decline (int), pattern (str | None -- "compression", "deceleration", "deterioration", or None).

    2. `compute_qoq(values: list[float | None]) -> list[float | None]` -- Sequential quarter-over-quarter % change. values[0] is most recent. First entry has no prior = None. For each pair, compute (curr - prev) / abs(prev) * 100. Handle None and zero gracefully.

    3. `compute_yoy(quarters: list[QuarterlyPeriod], concept: str, statement: str) -> list[float | None]` -- Same-quarter YoY change. Match by fiscal_quarter (e.g., Q1 FY2025 vs Q1 FY2024). This eliminates seasonality. Returns list aligned with quarters (most recent first). None when no matching prior year quarter.

    4. `compute_acceleration(qoq_changes: list[float | None]) -> float | None` -- Most recent QoQ change minus prior QoQ change. Positive = growth speeding up. Negative = growth slowing. None if insufficient data.

    5. `detect_sequential_pattern(qoq_changes: list[float | None], concept: str, threshold: int = 4) -> tuple[str | None, int]` -- Scan qoq_changes for threshold+ consecutive negative values. Returns (pattern_name, consecutive_count). Pattern name based on concept: "margin" in concept -> "compression", "growth" in concept -> "deceleration", else "deterioration". Returns (None, 0) if no pattern.

    6. `compute_trends(quarters: list[QuarterlyPeriod], concept: str, statement: str) -> TrendResult` -- Orchestrator: extracts values for concept from quarters, calls compute_qoq, compute_yoy, compute_acceleration, detect_sequential_pattern. Returns TrendResult.

    7. `compute_all_trends(quarterly: QuarterlyStatements) -> dict[str, TrendResult]` -- For every concept that has data in at least 2 quarters, run compute_trends. Returns dict keyed by concept name.

    Helper: `_extract_concept_values(quarters: list[QuarterlyPeriod], concept: str, statement: str) -> list[float | None]` -- Pull the .value from the SourcedValue for a concept across quarters. statement determines whether to look in .income, .balance, or .cash_flow dict.
