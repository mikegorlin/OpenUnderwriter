    2. Deferred revenue divergence:
       - rev_current, rev_prior = revenue latest and prior
       - deferred_current, deferred_prior = deferred_revenue latest and prior
       - rev_growth = safe_ratio(rev_current - rev_prior, rev_prior)
       - deferred_growth = safe_ratio(deferred_current - deferred_prior, deferred_prior)

       - ar_growth = safe_ratio(ar_current - ar_prior, ar_prior)
       - ratio = safe_ratio(ar_growth, rev_growth) if rev_growth > 0

       - Collect gross_margin (gross_profit / revenue) across all available periods
