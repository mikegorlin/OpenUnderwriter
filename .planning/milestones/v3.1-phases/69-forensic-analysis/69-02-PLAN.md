    - compute_capital_allocation_forensics(statements, market_data=None) returns (CapitalAllocationForensics, ExtractionReport)
    - ROIC = EBIT * (1 - ETR) / (equity + total_debt - cash). Zone: <5% = danger, <10% = warning, else safe. Trend from prior period comparison.
    - Acquisition effectiveness = goodwill_growth_rate vs revenue_growth_rate. If goodwill growing 2x+ revenue = danger, 1.5x = warning.
    - Buyback timing = share_repurchases / decrease_in_shares if computable; else None. Compare against avg stock price from market_data if available. Zone: implied price > 1.3x avg = danger (overpaying), > 1.1x = warning.
    - Dividend sustainability = FCF payout ratio (dividends_paid / (OCF - capex)). Zone: >100% = danger, >80% = warning, else safe.
    - None inputs produce insufficient_data zones
  </behavior>
  <action>
    Create `src/do_uw/stages/analyze/forensic_capital_alloc.py` (~200 lines):

    1. `compute_capital_allocation_forensics(statements: FinancialStatements, market_data: dict[str, Any] | None = None) -> tuple[CapitalAllocationForensics, ExtractionReport]`

    2. ROIC computation:
       - ebit = _extract_input(statements, "ebit") OR operating_income
       - tax_rate = safe_ratio(_extract_input(statements, "income_tax_expense"), _extract_input(statements, "pretax_income"))
       - nopat = ebit * (1 - (tax_rate or 0.21)) -- use statutory rate as fallback
       - invested_capital = equity + total_debt - cash
       - roic = safe_ratio(nopat, invested_capital)
       - Prior period ROIC for trend
       - Zone: <0.05 danger, <0.10 warning, else safe

    3. Acquisition effectiveness:
       - goodwill_growth = safe_ratio(gw_current - gw_prior, gw_prior)
       - rev_growth = safe_ratio(rev_current - rev_prior, rev_prior)
       - effectiveness = safe_ratio(goodwill_growth, rev_growth) if rev_growth > 0
       - Zone: ratio > 2.0 danger, > 1.5 warning, else safe
       - Note: acquisitions_net concept added by Phase 67 -- handle None gracefully

    4. Buyback timing quality:
       - repurchases = abs(_extract_input(statements, "share_repurchases") or 0)
       - If repurchases == 0: zone = "not_applicable"
       - shares_current = _extract_input(statements, "shares_outstanding")
       - shares_prior = _extract_input(statements, "shares_outstanding", "prior")
       - shares_retired = (shares_prior or 0) - (shares_current or 0)
       - implied_price = safe_ratio(repurchases, shares_retired) if shares_retired > 0
       - avg_price from market_data.get("avg_close") if available
       - premium = safe_ratio(implied_price, avg_price)
       - Zone: premium > 1.3 danger, > 1.1 warning, else safe

    5. Dividend sustainability:
       - dividends = abs(_extract_input(statements, "dividends_paid") or 0)
       - fcf = (ocf or 0) - abs(capex or 0) where ocf = operating_cash_flow, capex = capital_expenditures
       - payout_ratio = safe_ratio(dividends, fcf) if fcf > 0
       - Zone: > 1.0 danger, > 0.8 warning, else safe

    - Test dividend sustainability: payout 120% -> danger, 50% -> safe
    - Test all None inputs: all insufficient_data
