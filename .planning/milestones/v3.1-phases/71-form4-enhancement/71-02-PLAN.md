    ```python
    class FilingTimingSuspect(BaseModel):
        insider_name: str
        transaction_date: str
        transaction_type: str  # SELL or BUY
        filing_date: str  # 8-K filing date
        filing_item: str  # 8-K item number (e.g., "2.02")
    ```

    **Detection functions (insider_trading.py or insider_trading_analysis.py):**

    1. `detect_exercise_sell_patterns(transactions: list[InsiderTransaction]) -> list[ExerciseSellEvent]`:
       - Group transactions by (owner_name, date) with 1-day tolerance (T+0 or T+1)
       - For each group: find code "M" exercises and code "S" sells
       - If both exist: create ExerciseSellEvent with combined totals
       - Severity always AMBER per user decision
       - Inherit 10b5-1 status from the sell transaction

    2. `classify_8k_item(item_number: str) -> str`:
       - NEGATIVE: "2.02" (earnings), "5.02" (departure), "4.02" (non-reliance on financials)
       - POSITIVE: "1.01" (material agreement), "2.01" (acquisition)
       - NEUTRAL: everything else
       - Deterministic classification per user decision (no LLM)

    3. `analyze_filing_timing(transactions: list[InsiderTransaction], state: AnalysisState, window_days: int = 60) -> list[FilingTimingSuspect]`:
       - Get 8-K filing metadata from `state.acquired_data.filings.get("8-K", [])`
       - For each 8-K with classifiable item number:
         - Find SELL transactions within window_days BEFORE negative 8-K filings
         - Find BUY transactions within window_days BEFORE positive 8-K filings
       - Exclude gift/estate/compensation codes
       - Severity: sell-before-negative = RED_FLAG, buy-before-positive = AMBER
       - Use filing_date as conservative proxy per research (Pitfall 5)

    **Wire into extract_insider_trading():**
    After dedup and filtering (from Plan 01), call both detection functions. Store results on InsiderTradingAnalysis.

    **500-line limit:** If insider_trading.py exceeds 500 lines, extract pattern detection functions into `insider_trading_patterns.py` in the same directory. Keep the main file as the orchestrator; new file holds detect_exercise_sell_patterns, classify_8k_item, analyze_filing_timing.
