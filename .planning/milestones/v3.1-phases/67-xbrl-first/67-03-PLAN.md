   @dataclass
   class DerivedDef:
       name: str
       inputs: list[str]       # required primitive concept names
       compute: Callable       # function(inputs_dict) -> float | None
       statement: str          # which statement type this belongs to
       description: str
       expected_sign: str      # for consistency with config

   - Guards against zero denominators
   - Returns dict of derived concept name -> computed value
   - Logs computed values at DEBUG level

   - For revenue_growth_yoy: needs consecutive periods, computes (curr - prev) / abs(prev) * 100

   - Return None if denominator == 0
