**`_compute_direct_percentile(company_cik, frames_data, sic_mapping, company_sic, higher_is_better) -> FramesPercentileResult`**:
- Find company's value in frames_data by CIK
- If not found, return FramesPercentileResult with all None
- Extract all `val` values for overall percentile via `percentile_rank()`
- Filter to same 2-digit SIC prefix for sector percentile
- Return populated FramesPercentileResult

**`_compute_derived_percentile(company_cik, numerator_data, denominator_data, sic_mapping, company_sic, higher_is_better) -> FramesPercentileResult`**:
- Inner-join numerator and denominator by CIK
- Compute ratio per entity (skip division by zero)
- Find company's ratio
- Rank via percentile_rank() for overall and sector

**`compute_frames_percentiles(frames_data, company_cik, company_sic, sic_mapping) -> dict[str, FramesPercentileResult]`**:
- For each direct metric in FRAMES_METRICS (from sec_client_frames): compute direct percentile
- For each derived metric: compute derived percentile
- Return dict keyed by metric_name
- Skip metrics where frames_data is empty or missing
