## RATIONALE FOR SOURCE SELECTION

**Criteria used**:
1. **Relevance to D&O risk**: Does this source track developments that create or indicate D&O liability?
2. **Timeliness**: Does this source break news early (before mainstream media)?
3. **Signal-to-noise ratio**: Does this source provide actionable intelligence (not just noise)?
4. **Credibility**: Is this source authoritative and reliable?
5. **Accessibility**: Is this source publicly available or reasonably priced?

**Sources excluded**:
- Social media (too much noise, unreliable)
- Blogs without institutional backing (credibility concerns)
- Paywalled sources without clear ROI (e.g., $10K+ subscriptions)
- Sources that duplicate information already available from included sources
