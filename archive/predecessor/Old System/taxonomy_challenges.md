# Taxonomy Challenges & Edge Cases

## Purpose
This file tracks cases that don't fit cleanly into the existing hazard taxonomy, revealing gaps and opportunities for framework improvement.

## Tracking Criteria
- Cases that straddle multiple hazard categories
- Cases that reveal new risk patterns not captured in current taxonomy
- Cases where the classification feels forced or ambiguous
- Cases that suggest the need for new industry categories or sub-risks
