## FORMATTING GUIDELINES

### Mobile Optimization
- Use bullet points, not paragraphs
- Key-value pairs for quick scanning
- Tables only for structured data
- No wide tables (max 6 columns)
- Bold key terms only

### Content Standards
- Each bullet = one fact, one source
- Concerns ranked by severity
- Positives ranked by importance
- Recommendation in 2-3 sentences max
- No verbose explanations

### v1.2 Stock Section Requirements
- ALL horizons from STK-001 must be shown
- Attribution (STK-008) required if any horizon shows >10% decline
- Patterns (STK-010) must be checked and documented
- Sector comparison required for Short Interest (per SEC-008)

### Approval Flow
1. Analyst completes analysis
2. Routes based on authority trigger
3. VP reviews for standard referrals
4. SVP reviews for elevated/declined risks

---

## SECTION-BY-SECTION GUIDANCE

### Section 2: Summary & Recommendation
**Concerns**: Rank by litigation probability impact
- Lead with nuclear triggers if present
- Include STK-010 patterns if CASCADE or BREAKDOWN detected
- Include magnitude (dollar amounts, percentages)
- Focus on actionable risks

**Positives**: Rank by underwriting value
- Strong financials first
- Clean litigation history
- STK-008 SECTOR-WIDE or MARKET-WIDE attribution (mitigates stock decline concern)
- Governance strengths

**Recommendation**: Clear decision + rationale
- SUPPORT: "Write at proposed terms because..."
- SUPPORT WITH CONDITIONS: "Write subject to..."
- DECLINE: "Pass because..."

### Section 5: Stock & Market (v1.2 Enhanced)
**Multi-Horizon Table Required**:
- Show all STK-002 through STK-007 horizons
- Include sector comparison for each
- Show attribution classification
- Color-code severity (ðŸŸ¢/ðŸŸ¡/ðŸ”´)

**Attribution Required** if ANY horizon shows >10% decline:
- Calculate via STK-008 methodology
- Classify as COMPANY-SPECIFIC, SECTOR-WIDE, or MARKET-WIDE
- Document the percentage point difference

**Pattern Flags Required**:
- Check all STK-010 patterns
- Document Y/N for each
- If Y, provide brief detail

### Section 8: Litigation & Claims
**Always include**:
- Stanford SCAC search date
- All prior securities matters
- Current litigation status
- Claims on Liberty policy

### Section 9: Forward Look
**Focus on actionable triggers**:
- Earnings dates with miss risk
- FDA/regulatory decisions
- Contract renewals
- Debt maturities

### Section 10: Scoring Detail (v1.2 NEW)
**Required for v1.2**:
- Show all 10 factors with scores
- F.2 must reference STK-007/008 output
- Show key driver for each factor
- Calculate and display total
